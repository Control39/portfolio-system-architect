feat(mcp): создать полноценный MCP сервер для Career Autopilot

Архитектура MCP сервера:

Инструменты:
- file_tools.py - работа с файлами проекта
- git_tools.py - Git операции и SourceCraft
- compass_tools.py - IT-Compass маркеры (83 маркера в 19 доменах)
- chroma_tools.py - векторный поиск ChromaDB
- monitoring_tools.py - Prometheus, Grafana, Docker stats
- main.py - app management (12 микросервисов)
- main.py - security tools (bandit, pip-audit)

Ресурсы:
- navigate://{audience} - навигация для HR, tech, grant, community
- it-compass://domain/{domain} - информация о доменах

Интеграция:
- mcp-config.json - конфигурация для MCP клиентов
- Dockerfile - контейнеризация
- requirements.txt - зависимости
- README.md - полная документация

Сценарии использования:
1. Koda AI assistant - автоматизация разработки
2. Claude Desktop - работа с кодом и документами
3. SourceCraft - грантовая заявка и интеграция
4. CI/CD - сканирование безопасности и мониторинг
