# Как использовать IT Compass для трудоустройства
## 🎯 Когда можно идти на junior-собеседование?
| Направление | Минимум для «готов к тех. собесу» |
|-------------|----------------------------------|
| Python      | `python_1_1` (CSV-скрипт) |
| Docker      | `docker_1_1` (Dockerfile + контейнер) |
| Git/GitHub  | хотя бы 1 публичный репозиторий с коммитами |
| Общее       | ≥3 high-priority маркеров |
## 💬 Что говорить на собеседовании:
> *«Я не оцениваю себя — я подтверждаю навыки действиями.
> Вот мой прогресс: [ссылка на my_portfolio.md].
> Готов углубиться в любой из этих кейсов: от CSV-скрипта до MLflow-логирования».*
## 📎 Куда прикладывать:
- В сопроводительное письмо — ссылку на `my_portfolio.md`
- В Telegram/LinkedIn — краткую версию:
*«Мой IT-рост: [скрин] + [ссылка]»*
- На тех. собесе — предложи:
*«Хотите, я покажу, как работает мой Dockerfile?»*
