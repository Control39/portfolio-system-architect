# .continue/

Конфигурации для Continue - инструмента разработки с ИИ.

## Назначение
- Интеграция с IDE (VS Code и др.) через Continue
- Управление поведением ИИ-ассистента в редакторе кода
- Определение агентов, которые будут доступны в Continue

## Структура
- `agents/` - конфигурации специализированных агентов для Continue

Этот каталог обеспечивает интеграцию всей системы с вашей средой разработки, позволяя использовать продвинутые ИИ-функции непосредственно в редакторе.
