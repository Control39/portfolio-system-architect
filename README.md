# Система управления карьерным развитием

## Описание проекта

Система управления карьерным развитием на основе методологии "Объективные маркеры компетенций". Позволяет отслеживать и развивать профессиональные навыки сотрудников с помощью объективных метрик.

## Роль в экосистеме

Career Development System является приложением в экосистеме Cognitive-Architecture, которое предоставляет интерфейс для отслеживания карьерного развития и интеграции данных из других компонентов экосистемы.

## Ключевые особенности

- Отслеживание профессиональных навыков с помощью объективных метрик
- Интеграция с IT-Compass для получения данных о компетенциях
- Интеграция с Portfolio Organizer для получения материалов
- Веб-интерфейс, мобильное приложение и API для управления развитием

## Технологический стек

- Python (FastAPI) для API
- JavaScript (React) для веб-интерфейса
- JavaScript (React Native) для мобильного приложения
