# 🧠 portfolio-system-architect

> **Документальный фильм о рождении новой профессии: Архитектор когнитивных систем**

Этот репозиторий — не просто код, а **повествование, разбитое на пять глав**, где личный путь от хаоса к порядку становится доказательством концепции новой профессиональной роли.

Проект подаётся на грант **SourceCraft Open Source**.

---

## 🎯 Зачем это всё?

Я провела 2.5 года в поиске ответа на вопрос: *«Кто я в IT, если я не кодер?»*

Этот репозиторий — мой ответ. Я создала **систему для проектирования профессионального пути архитектора когнитивных систем** — специалиста, который:
- проектирует системы, где человек управляет ИИ;
- создаёт новые формы экспертизы через оркестрацию технологий;
- ценит системное мышление выше написания кода.

---

## 🧠 Что внутри?

| Компонент | Что делает |
|-----------|------------|
| **IT-Compass Framework** | Система навигации по профессиональному развитию на основе объективных маркеров |
| **RAG-система** | Автоматически извлекает достижения из тысяч файлов диалогов и заметок |
| **Arch-Compass** | Архитектурный фреймворк для проектирования систем с ИИ |
| **ML Model Registry** | Полноценный реестр моделей с 11 видами тестов (включая fuzz и безопасность) |
| **Career Development System** | Мобильное и веб-приложение для трекинга компетенций |
| **8 аналитических кейсов** | Uber Freight, Брусника, Patreon/Apple и другие — с количественными моделями |

## 🔬 Доказательство концепции

> *«За два с половиной года я прошла путь от Excel-таблички до архитектора когнитивных систем. Я не пишу код — я проектирую системы, в которых ИИ становится моим агентом. Моя работа — это вклад в будущее профессии, где ценность создаётся через системное мышление и оркестрацию технологий.»*

**Ключевые метрики:**
- ✅ **Экосистема из 8 компонентов**, связанных в единое целое
- ✅ **Сотни уникальных файлов** — ни одного дубликата (автоматическая верификация)
- ✅ **11 тестов** для ML Model Registry (контракты, fuzz, безопасность)
- ✅ **7 ADR-документов** с архитектурными решениями
- ✅ **Полностью автоматизированная генерация** сайта и карты знаний
