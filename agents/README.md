## Назначение

Эта директория содержит агентов экосистемы. В настоящее время используется только **один центральный агент** - `cognitive_agent`.

## 🧠 Cognitive Agent (центральный)

Это единственный агент в экосистеме - центральный оркестратор, который:

- 🔍 **Сканирует** все сервисы и понимает их структуру
- 📋 **Планирует** задачи через ИИ (LangChain + GigaChat/Ollama)
- 🔗 **Интегрирует** IT-Compass, Decision Engine, Job Automation
- 📚 **Собирает метрики** и учится на результатах
- 🎯 **Выполняет** навыки (skills) через скрипты
- 📈 **Анализирует** качество кода, документации и тестов
- 🔐 **Обеспечивает** enterprise-уровень безопасности

### Структура cognitive_agent

- `autonomous_agent_enterprise.py` - Главный агент (Enterprise версия)
- `autonomous_agent.py` - Базовая версия агента
- `src/` - Исходный код базовых компонентов
- `common/` - Общие утилиты и базовые классы
- `skills/` - Наборы навыков агента (25+ навыков)
- `config/` - Конфигурационные файлы
- `tests/` - Тесты для компонентов агента
- `docs/` - Документация по агенту
- `chroma_db/` - Векторная база данных для RAG
- `logs/` - Журналы работы агента

### Функциональные возможности

1. **Анализ качества кода**:
   - Проверка типизации с помощью MyPy
   - Проверка стиля кода с помощью Ruff
   - Анализ безопасности с помощью Bandit
   - Проверка типов с помощью Pyright
   - Оценка покрытия тестами

2. **Анализ документации**:
   - Проверка наличия docstring'ов
   - Анализ структуры Markdown файлов
   - Оценка согласованности между кодом и документацией

3. **Анализ тестов**:
   - Поиск файлов тестов
   - Определение используемого фреймворка
   - Оценка качества и покрытия тестами

## Безопасность

Все агенты работают в соответствии с системой enterprise-контроля доступа, которая обеспечивает:

- Проверку разрешений на выполнение операций
- Защиту от небезопасных команд
- Аудит выполненных действий
- Контроль использования ресурсов

Для получения подробной информации о системе безопасности см. [enterprise_guardrails.py](./cognitive_agent/enterprise_guardrails.py).
