# Директория skills для Cognitive Agent

Эта директория содержит навыки (skills), которые Cognitive Agent может использовать для выполнения различных задач.

## Структура

Директория содержит различные навыки, организованные по категориям:

- [code_analysis/](./code_analysis) - Навыки для анализа кода
- [documentation/](./documentation) - Навыки для работы с документацией
- [testing/](./testing) - Навыки для работы с тестами
- [project_management/](./project_management) - Навыки для управления проектами
- [security/](./security) - Навыки для анализа безопасности
- [other/](./other) - Прочие навыки

## Основные категории навыков

### Анализ кода

Навыки в категории [code_analysis/](./code_analysis) позволяют агенту:
- Проверять качество кода с использованием различных инструментов
- Выявлять потенциальные ошибки и уязвимости
- Анализировать структуру кода и зависимости

### Документация

Навыки в категории [documentation/](./documentation) позволяют агенту:
- Проверять полноту и актуальность документации
- Обновлять документацию в соответствии с изменениями в коде
- Генерировать документацию для новых компонентов

### Тестирование

Навыки в категории [testing/](./testing) позволяют агенту:
- Анализировать существующие тесты
- Генерировать новые тесты
- Оценивать покрытие кода тестами

## Создание новых навыков

Чтобы создать новый навык:
1. Создайте новую поддиректорию в соответствующей категории
2. Добавьте файл main.py с реализацией навыка
3. Добавьте файл SKILL.md с описанием навыка
4. Убедитесь, что навык соответствует интерфейсу, определенному в базовом классе агента

## Использование

Агент автоматически обнаруживает и использует навыки из этой директории при выполнении задач.
