```
├── logs/                    # Логи агентов
│   ├── alerts.log          # Оповещения

│   └── scheduled-monitor.log  # Плановые
│
├── data/
│   └── trigger_metrics.db  # База событий
│
└── config/
    └── agent-config.yaml   # Конфигурация

agents/cognitive_agent/
├── logs/                   # Логи компонентов
│   ├── scanner.log
│   ├── planner.log
│   └── triggers.log
│
├── data/
│   └── learning/
│       ├── metrics/       # Метрики задач

└── config/               # Конфиги компонентов
```

## 🔄 Циклы работы

### Автоматический запуск (каждые 6 часов)

```mermaid
timeline
    title Цикл работы агента
    section 00:00
        Запуск по расписанию
        Сканирование проекта
    section 00:05
        Планирование задач
        Запуск навыков
    section 00:30
        Сбор метрик
        Генерация отчётов
    section 01:00
        Оповещения (если есть проблемы)
        Завершение цикла
```

### Ручной запуск (по триггеру)

```mermaid
sequenceDiagram
    User->>Trigger: Событие
    Trigger->>Processor: Обработка
    Processor->>Scanner: Запуск сканирования
    Scanner->>Planner: Результаты
    Planner->>Skills: План задач
    Skills->>User: Результат выполнения
```
