# Руководство по вкладу в Манифест

Спасибо, что хотите внести свой вклад в манифест `portfolio-system-architect`! Этот документ описывает процесс внесения изменений в репозиторий, который является документальным фильмом о рождении новой профессии: архитектора когнитивных систем.

Каждый ваш вклад- это кирпичик в здании новой профессиональной реальности, где ценность создается через системное мышление и оркестрацию ИИ, а не через написание кода.

## Как внести вклад

Ваш вклад- это не просто исправление багов, а участие в создании нового повествования о будущем профессий. Вы можете внести вклад тремя способами:

### 1. Участие в общем манифесте

Создайте свой форк репозитория и внесите изменения в основную структуру:

```bash
git clone https://github.com/Control39/portfolio-system-architect.git
cd portfolio-system-architect
```

Создайте ветку для вашей функции:

```bash
git checkout -b feature/ваша-фича
```

Внесите изменения и зафиксируйте их:

```bash
git add .
git commit -m "Добавил: ваше описание изменений"
```

Отправьте изменения в ваш форк:

```bash
git push origin feature/ваша-фича
```

Создайте Pull Request в оригинальный репозиторий.

### 2. Создание вашего собственного манифеста

Создайте форк репозитория и используйте его как шаблон для вашего собственного пути. Вы можете:
- Рассказать свою историю в структуре "Путь--> Методология--> Доказательства--> Артефакты--> Манифест"
- Создать свою версию IT-Compass для вашей области
- Опубликовать ваш манифест и поделиться им с миром

### 3. Прямое участие в диалоге

Если вы не хотите создавать Pull Request, вы все равно можете внести вклад:
- Создайте issue с вашей идеей, вопросом или историей
- Напишите мне напрямую на leadarchitect.ai@gmail.com
- Поделитесь, как манифест повлиял на вас и ваш путь

## Стиль вклада

Ваш вклад может быть любого вида, главное- искренность и подлинность:

- Пишите в стиле, который отражает ваш внутренний голос
- Используйте Markdown для форматирования текста
- Добавляйте примеры из вашего опыта
- Делитесь не только успехами, но и вызовами на пути

Главное правило: ваш вклад- это не код, а часть повествования о будущем профессий в эпоху ИИ.

## Повествование

Ваш вклад- это часть большого повествования о будущем профессий. При создании вклада:

- Используйте Markdown для форматирования текста
- Добавляйте примеры из вашего личного опыта и пути
- Рассказывайте истории, а не просто перечисляйте факты
- Делитесь не только успехами, но и вызовами на пути

Главное- создавать живое, искреннее повествование, которое вдохновляет других.

## Аутентичность

При создании вклада помните:

- Не пытайтесь соответствовать ожиданиям
- Делитесь тем, что реально происходило с вами
- Ваш путь может быть непохож на мой и это прекрасно
- Главное- быть честным перед собой и другими

Повествование о будущем профессий строится на аутентичности, а не на идеализации.

## Сообщения о вкладе

Когда вы делитесь своим вкладом:

- Пишите кратко и искренне
- Указывайте, что именно вы хотите внести в повествование
- Делитесь своими чувствами и размышлениями

Примеры:
- "Добавить: моя история о том, как я осознал свою ценность за пределами написания кода"
- "Поделиться: как манифест повлиял на мой путь и помог переосмыслить свою роль"
- "Предложить: идею для нового раздела о будущем профессий в эпоху ИИ"

## Этика участия

При участии в создании манифеста:

- Уважайте путь каждого участника
- Цените разнообразие опытов и взглядов
- Поддерживайте атмосферу доверия и открытости
- Помните, что мы создаем будущее вместе

Более подробно см. [Кодекс поведения](CODE_OF_CONDUCT.md).
