# postgres/

Конфигурации и скрипты для PostgreSQL.

## Назначение
- Хранение конфигураций базы данных PostgreSQL
- Инициализация и настройка БД
- Скрипты для начальной загрузки данных

## Структура
- `init/` - скрипты инициализации базы данных при первом запуске

Этот каталог содержит все, что связано с базой данных PostgreSQL, включая настройки, скрипты инициализации и конфигурации, необходимые для корректной работы приложения с СУБД.
