# .codeassistant/

Конфигурации и правила для Code Assistant.

## Назначение
- Хранение правил и навыков для ассистента кодирования
- Управление поведением ассистента в различных сценариях
- Обеспечение согласованности в подходах к кодированию и анализу

## Структура
- `rules/` - правила анализа кода и архитектуры
- `skills/` - специализированные навыки (аудит безопасности, анализ репозитория и т.д.)
- `teacher/` - обучающие материалы, шаблоны, руководства
- `tools/` - вспомогательные инструменты для ассистента

Этот каталог определяет "личность" и компетенции вашего кодового ассистента, задавая стандарты качества и подходы к решению задач.
