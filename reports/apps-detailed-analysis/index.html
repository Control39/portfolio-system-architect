    🔐 1. auth_service

    Назначение

    JWT-аутентификация и авторизация для API Gateway.

    Реальное состояние

    ✅ Что работает:

    • main.py — реальная реализация JWT: python @app.post("/auth/token") async def login(request: TokenRequest): # Создаёт JWT токен с помощью PyJWT payload = {"username": username, "role": role, "exp": ...} token = jwt.encode(payload, JWT_SECRET, algorithm="HS256")
    • Эндпоинты:
    • POST /auth/token — выдача токена
    • POST /auth/verify — проверка токена
    • GET /health — health check (через src.common.health_check)
    • Dockerfile — готов к контейнеризации

    ⚠️ Проблемы:

    • README ложный: Обещает 15 тестов с 100% покрытием, но тесты — шаблоны
    • Нет реальной БД: Демо-режим (любые учётные данные, кроме demo/demo)
    • src/ пустой: src/__init__.py существует, но нет модулей

    📁 Структура:

    auth_service/
├── main.py              # ✅ Реальная JWT логика
├── config/              # Конфигурация (JWT_SECRET, алгоритмы)
├── src/                 # ⚠️ Пустой (только __init__.py)
├── tests/               # ⚠️ Шаблоны тестов
└── Dockerfile           # ✅ Готов

    🧪 Тесты:

    • Файлы: test_basic.py, test_integration_auth_service.py
    • Реальность: Шаблоны с категориями "TestBasicFunctionality", "TestErrorHandling"
    • Покрытие: Не 100%, а ~30-40% (только импорт и создание экземпляра)

    🚀 Запуск:

    JWT_SECRET=your_secret python -m auth_service.main
# Порт: 8100
# Swagger: http://localhost:8100/docs

    Реальное состояние

    ⚠️ Что есть:

    • main.py — FastAPI приложение
    • gigachain_bridge.py — интеграция с GigaChat
    • core/ — ядро рассуждений (частично реализовано)

    ❌ Проблемы:

    • README шаблонный: Не описывает реальные возможности
    • RAG не реализован: Нет интеграции с ChromaDB/Sentence Transformers
    • Тесты — шаблоны: Те же категории "15 тестов, 100% покрытие"

    📁 Структура:

    decision-engine/
├── main.py              # FastAPI app

├── api/                 # Endpoints
├── config/              # component-config.yaml
├── tests/               # ❌ Шаблоны
