# 📊 Примеры бейджей и метрик

> **Файлы в этой папке — шаблоны для документации, а не актуальные данные.**

## Назначение

Эта папка содержит примеры разметки и структуры данных, которые используются в туториалах,
тестах и как референс для создания реальных бейджей.

## Файлы

### `coverage.md`

Пример Markdown-разметки для бейджа покрытия тестами.

⚠️ **Не копируйте эту ссылку в README проекта** — используйте динамический бейдж из CI:

```markdown
[![Coverage](https://img.shields.io/codecov/c/github/Control39/portfolio-system-architect?logo=codecov&style=flat-square)](https://app.codecov.io/github/Control39/portfolio-system-architect)
```

Или запускайте `scripts/update-coverage-badge.py` для генерации актуального бейджа.

### `metrics.json`

Пример структуры JSON-отчёта о метриках. Используется в:
- Туториалах по формату данных
- Тестах скриптов обновления бейджей
- Документации как схема данных

## Правила использования

- ✅ Копируйте структуру для создания новых бейджей
- ✅ Используйте как референс для тестов
- ❌ Не ссылайтесь на эти файлы из `README.md` как на актуальные данные
- ❌ Не редактируйте эти файлы для обновления реальных метрик

## Актуальные метрики

Реальные данные о покрытии и тестах:
- `docs/TEST-COVERAGE-METRICS.md` — ручной отчёт
- `scripts/update-coverage-badge.py` — автоматическое обновление бейджа в README
