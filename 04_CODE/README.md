# Исходный код IT‑Compass

Этот раздел содержит исходный код модулей IT‑Compass, интегрированных в общую архитектуру portfolio‑system‑architect.

## Структура

```
04_CODE/
├── src/                          # Исходный код
│   ├── core/                     # Основные модули
│   │   ├── tracker.py            # Трекер компетенций
│   │   └── mental/               # Модули психологической поддержки
│   │       ├── support.py        # Объединённый модуль поддержки
│   │       └── low_energy.py     # Режим низкой энергии (опционально)
│   └── cli/                      # Интерфейс командной строки
│       └── main.py               # Точка входа CLI
├── tests/                        # Модульные и интеграционные тесты
│   ├── test_tracker.py
│   ├── test_support.py
│   └── test_cli.py
└── config/                       # Конфигурационные файлы
    └── settings.json
```

## Перенос кода

Код будет постепенно перенесён из `components/it-compass/` в эту структуру, чтобы обеспечить единообразие и упростить интеграцию.

## Требования

- Python 3.8+
- Зависимости перечислены в `requirements.txt` корневого каталога

## Запуск

```bash
cd 04_CODE
python src/cli/main.py --help
```

## Тестирование

```bash
pytest tests/
```
