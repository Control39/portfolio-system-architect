# ADR-001: Выбор методологии системного мышления

## Статус
Принято

## Контекст
Для создания эффективной архитектуры портфолио системы архитектора необходимо выбрать подходящую методологию, которая позволит структурировать мышление и подход к решению архитектурных задач. Необходимо определить основные принципы, которые будут лежать в основе всей системы.

## Рассмотренные варианты
1. **Классическое системное мышление** - традиционный подход к анализу систем
2. **Soft Systems Methodology (SSM)** - методология мягких систем Чекленда
3. **System Dynamics** - динамика систем
4. **Viable System Model (VSM)** - модель жизнеспособной системы

## Решение
Выбрана комбинированная методология, основанная на принципах системного мышления с элементами Soft Systems Methodology. Эта методология позволяет:

- Определять сложные проблемы в терминах систем
- Использовать концепцию "систем мышления" для анализа
- Применять богатую картину (rich picture) для визуализации проблем
- Использовать концептуальную модель для структурирования решений

## Последствия
### Положительные:
- Повышение структурированности подхода к решению архитектурных задач
- Улучшение понимания сложных взаимосвязей в системе
- Развитие системного мышления как ключевой компетенции

### Отрицательные:
- Необходимость обучения методологии
- Потенциальная сложность применения для новых участников
