# legacy/

> Архив устаревших версий модулей. Сохранена полная история коммитов (`git mv`).

## Содержимое

| Папка | Что это | Почему в архиве | Активная версия |
|-------|---------|-----------------|-----------------|
| `decision_engine_v1/` | Устаревшая версия reasoning-движка | Не импортировалась никем из `apps/`, дублировала `apps/decision-engine/` | `apps/decision-engine/` |
| `repo_audit_v1/` | Устаревшая версия инструмента аудита | 0 импортёров, дублировала `tools/repo_audit/` | `tools/repo_audit/` |
| `test_decision_engine_init_v1.py` | Тесты для `src/decision_engine/` | Тестировал удалённый (перемещённый) код | Тесты `apps/decision-engine/` |

## Правила работы с архивом

- **Не импортировать** код из `legacy/` — это историческая справка
- **Не модифицировать** — если нужно что-то взять, скопируйте в активную папку
- **Не удалять** без согласования — история коммитов связана с этими путями

## Когда обращаться к архиву

- Поиск документов или контекста по старым версиям
- Восстановление кода при ошибочном удалении
- Аудит истории изменений (`git log --follow legacy/decision_engine_v1/`)
