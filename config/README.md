# config/

Центральные конфигурации проекта.

## Назначение
- Хранение общих конфигурационных файлов
- Централизованное управление настройками для разных частей системы
- Разделение конфигураций по типам (AI, VS Code и др.)

## Структура
- `ai/` - конфигурации для ИИ-компонентов и агентов
- `vscode/` - конфигурации для Visual Studio Code

Этот каталог предоставляет единое место для хранения конфигураций, которые могут использоваться несколькими приложениями или компонентами системы, обеспечивая согласованность и упрощая управление настройками.
