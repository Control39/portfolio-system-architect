# apps/

Микросервисы и приложения проекта.

## Назначение
- Хранение отдельных приложений и сервисов
- Изоляция логики различных компонентов системы
- Поддержка архитектуры на основе микросервисов

## Приложения
- `ai-config-manager/` - менеджер конфигураций ИИ
- `auth_service/` - сервис аутентификации и авторизации
- `career_development/` - сервис развития карьеры
- `decision-engine/` - движок принятия решений
- `infra-orchestrator/` - оркестратор инфраструктуры
- `it_compass/` - IT Compass (основное приложение)
- `job-automation-agent/` - агент автоматизации задач
- `knowledge-graph/` - граф знаний
- `mcp-server/` - MCP сервер для ИИ-агентов
- `ml-model-registry/` - реестр ML моделей
- `portfolio_organizer/` - организатор портфолио
- `system-proof/` - система доказательств
- `template-service/` - сервис шаблонов
- `thought-architecture/` - архитектура мышления

Каждое приложение является независимым компонентом с собственной логикой, API и тестами, что обеспечивает модульность и возможность независимого развертывания.
