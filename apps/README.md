Порты (8001, 8002 и т.д.) используются **внутри** Docker network для маршрутизации через Traefik.\n\n## Зависимости от `src/`\n\nМикросервисы импортируют общие компоненты из корневого `src/`:\n\n```python\n# Пример из apps/auth_service/main.py\nfrom src.common.health_check import init_health_checks\n\n# Пример из apps/ml-model-registry/src/main.py\nfrom src.common.health_check import init_health_checks\nfrom src.common.async_helpers import fetch_parallel\n```\n\n## Внутренняя структура сервиса\n\nКаждый сервис может иметь **собственный** `src/` — это его внутренняя организация, не связанная с корневым `src/`:\n\n```\napps/template-service/\n├── src/              ← внутренний src сервиса\n│   ├── api/\n│   └── core/\n├── Dockerfile\n└── requirements.txt\n```\n\n> **Важно:** импорты вида `from src.api import router` внутри `apps/*/src/main.py` относятся к **внутреннему** `src` сервиса.\n\n
