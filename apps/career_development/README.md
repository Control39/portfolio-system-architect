# Career Development System

Это система управления карьерным развитием на основе объективных маркеров компетенций.

## Структура проекта

- `src/` - Исходный код (API, ядро, утилиты)
- `src/api/` - API сервисы (FastAPI)
- `src/core/` - Ядро системы (бизнес-логика)
- `src/utils/` - Вспомогательные модули
- `tests/` - Тесты
- `docs/` - Документация

## Начало работы

1. Установите зависимости
2. Запустите миграции базы данных
3. Запустите сервер разработки
