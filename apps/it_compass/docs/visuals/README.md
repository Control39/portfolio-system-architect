# Визуальные материалы IT Compass

В этой папке собраны визуальные материалы для проекта IT Compass — системы объективного отслеживания карьерного роста в IT.

## Содержание

### 1. Диаграммы архитектуры
- `overview.mmd` — общая архитектура системы
- `workflow.mmd` — workflow пользователя
- `data-flow.mmd` — поток данных между компонентами

### 2. Инфографика
- `markers-infographic.md` — инфографика по маркерам компетенций
- `methodology-visual.md` — визуализация методологии "Объективные маркеры компетенций"

### 3. Презентации
- `presentation.md` — простая презентация для новичков (3-5 слайдов)
- `pitch-deck.md` — презентация для инвесторов/партнеров

### 4. Схемы
- `kanban-board.md` — канбан-доска проекта в формате Markdown
- `roadmap.mmd` — дорожная карта развития проекта

## Как использовать

1. **Для разработчиков**: Используйте диаграммы для понимания архитектуры
2. **Для пользователей**: Покажите инфографику и презентации для быстрого ознакомления
3. **Для презентаций**: Используйте слайды для демонстрации проекта

## Форматы

- **Mermaid (.mmd)**: Диаграммы, которые можно отобразить в GitHub/GitLab
- **Markdown (.md)**: Текстовые описания с возможностью встраивания Mermaid
- **Изображения (.png/.svg)**: Готовые изображения (будут добавлены позже)

## Генерация изображений

Для преобразования Mermaid в изображения можно использовать:
- [Mermaid CLI](https://github.com/mermaid-js/mermaid-cli)
- Онлайн-редактор [Mermaid Live Editor](https://mermaid.live/)
