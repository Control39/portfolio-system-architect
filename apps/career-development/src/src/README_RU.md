# Система управления карьерным развитием

Это система управления карьерным развитием на основе объективных маркеров компетенций.

## Структура проекта

- `api/` - API сервисы
- `web/` - Веб-интерфейс
- `mobile/` - Мобильное приложение
- `core/` - Ядро системы
- `utils/` - Вспомогательные модули
- `tests/` - Тесты
- `docs/` - Документация

## Начало работы

1. Установите зависимости для API:
   ```
   pip install -r api/requirements.txt
   ```

2. Запустите API сервер:
   ```
   python api/app.py
   ```

3. Откройте `web/index.html` в браузере для использования веб-интерфейса

## Архитектура системы

### Компоненты

1. **API Сервисы** - RESTful API для управления пользователями, навыками и маркерами компетенций
2. **Веб-интерфейс** - Клиентская часть для взаимодействия с системой через браузер
3. **Мобильное приложение** - Мобильный клиент для отслеживания прогресса
4. **Ядро системы** - Бизнес-логика и модели данных
5. **Вспомогательные модули** - Утилиты и хелперы

### Модели данных

- **Пользователь (User)** - Представляет сотрудника в системе
- **Навык (Skill)** - Конкретный навык пользователя с уровнем владения
- **Маркер компетенции (CompetencyMarker)** - Объективный маркер достижения определенного уровня компетенции
- **Запись прогресса (ProgressRecord)** - История изменений навыков пользователя

## Методология

Система основана на методологии "Объективные маркеры компетенций", которая позволяет:

1. Определить четкие критерии достижения компетенций
2. Отслеживать прогресс в развитии навыков
3. Планировать карьерный рост на основе объективных данных
4. Сравнивать уровень компетенций между сотрудниками
