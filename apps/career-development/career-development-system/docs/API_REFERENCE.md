# API Reference

## Endpoints

| Method | Path | Описание |
|--------|------|----------|
| **GET** | `/` | Тестовый запрос – проверка, что сервер жив |
| **GET** | `/profile` | Возвращает текущий профиль пользователя |
| **POST** | `/goals` | Создаёт новую карьерную цель |
| **PATCH** | `/markers/{id}` | Обновляет статус маркера компетенции |
| **GET** | `/evidence/export` | Генерирует пакет доказательств (PDF/Markdown) |
| **GET** | `/skills` | Список всех навыков и их уровней |
| **POST** | `/skills` | Добавление нового навыка |
| **GET** | `/progress` | Возвращает процент завершения всех маркеров |
