# 🧠 IT COMPASS: РУКОВОДСТВО ПО ПСИХОЛОГИЧЕСКОЙ ПОДДЕРЖКЕ

## 🎯 Цель раздела
Этот раздел создан для того, чтобы помочь пользователям IT Compass справляться с эмоциональными и психологическими аспектами обучения и профессионального развития в IT.
Я испытала на себе, знаю что путь в IT может быть сложным и предлагаю инструменты для поддержки психологического благополучия.

## 🚀 Как использовать раздел поддержки

### Быстрый доступ

```python
src/main.py --mental-support
```

### Основные функции:

* 💬 **Мотивационные цитаты** - ежедневная поддержка
* 🆘 **Кризисные номера помощи** - экстренная поддержка
* 🧘 **Режим 'низкой энергии'** - для периодов усталости
* 👥 **Руководство сообщества** - помощь в построении здоровых отношений

## 🧩 Интеграция с основным приложением

### Случайные мотивационные сообщения

При запуске приложения автоматически показывается случайная цитата. Это можно настроить в файле `config/settings.json`:

```json
{
    "show_motivational_quotes": true,
    "quote_frequency": "daily"
}
```
### Ежедневная мотивация

Каждое утро при первом запуске приложения показывается специальная мотивация на день.

### Режим 'низкой энергии'

Активируется автоматически при обнаружении длительных периодов неактивности или по команде:

```
bash
python src/main.py --low-energy
```

## 🔌 Интеграция с API сервисом

### Доступные сервисы:

1. **Психологи России** - поиск специалистов по городу
2. **Mindful** - медитации для снятия стресса
3. **Психологическая помощь** - горячие линии

### Как настроить:

1. Создайте файл `.env` в корне проекта
2. Добавьте API ключи:

```env
PSYHELP_API_KEY=your_api_key_here
MINDFUL_API_KEY=your_api_key_here
```

## 🤗 Community Guide

### Принципы здорового сообщества:

* **Взаимное уважение** - ценим разные точки зрения
* **Поддержка без осуждения** - помогаем, а не критикуем
* **Безопасное пространство** - уважаем личные границы
* **Рост через обратную связь** - конструктивная критика

### Как помочь коллеге:

1. **Выслушайте, не перебивайте**
2. **Спрашивайте** какая помощь нужна
3. **Предлагайте ресурсы**, а не решения
4. **Поощряйте** маленькие победы

## ⚠️ Важные предупреждения

### Что НЕ является заменой профессиональной помощи:

* Советы от AI или сообщества
* Самодиагностика по симптомам
* Самолечение

### Когда обратиться к специалисту:

* Постоянное чувство тревоги или подавленности
* Проблемы со сном более 2 недель
* Потеря интереса к деятельности, которая раньше приносила удовольствие
* Мысли о самоповреждении

## 🔧 Технические детали

### Структура файлов:

* `support/` - основная папка раздела
