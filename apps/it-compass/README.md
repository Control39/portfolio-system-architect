# IT-Compass

Объединённая версия методологии и приложения IT-Compass.

## Назначение

IT-Compass — это интегрированная система для:
- Отслеживания прогресса в изучении IT-направлений
- Интеграции с внешними API
- Психологической поддержки разработчиков
- Генерации портфолио
- Управления карьерой

## Структура компонента

```
it-compass/
├── config/                     # Конфигурационные файлы
├── docs/                       # Документация
│   ├── ARCHITECTURE.md         # Архитектура компонента
│   ├── METHODOLOGY.md          # Методология
│   ├── PROJECT_ANALYSIS.md     # Анализ проекта
│   ├── SUPPORT_GUIDE.md        # Руководство по поддержке
│   ├── PSYCHOLOGICAL_SUPPORT.md # Психологическая поддержка
│   └── context_for_ai_analysis.md # Контекст для анализа ИИ
├── examples/                   # Примеры использования
├── scripts/                    # Вспомогательные скрипты
├── src/                        # Исходный код
│   ├── main.py                 # Точка входа
│   ├── core/                   # Основная логика
│   ├── data/                   # Данные и маркеры
│   ├── ui/                     # Веб-интерфейс
│   └── utils/                  # Утилиты
├── support/                    # Ресурсы поддержки
├── tests/                      # Тесты
├── requirements.txt            # Зависимости Python
└── README.md                  # Этот файл
```

## Использование

Для запуска приложения:

```bash
python src/main.py
```

Для запуска тестов:

```bash
pytest tests/
```
