## Cloud Reason (RAG): Пример для новичка

### Задача
Понять, как ИИ помогает в архитектурном анализе.

### Действие
1. Изучить демо-пример в `components/cloud_reason/examples/basic_analysis.py`
2. Запустить демонстрацию:
   ```bash
   cd components/cloud_reason
   python examples/demo_analysis.py
   ```
3. Проанализировать результаты в консоли и в `reports/demo/`

### Результат
Практическое понимание работы RAG-системы и её применения в архитектурном анализе.
