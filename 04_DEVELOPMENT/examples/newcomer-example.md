## Arch-Compass: Пример для новичка

### Задача
Понять принципы проектирования архитектуры веб-приложения.

### Действие
1. Изучить примеры архитектур в `components/arch-compass-framework/examples/`
2. Пройти интерактивное руководство:
   ```bash
   cd components/arch-compass-framework
   python src/cli.py tutorial web-app
   ```
3. Создать свою первую архитектуру по шаблону:
   ```bash
   python src/cli.py init --template web-app my-project
   ```

### Результат
Понимание базовых принципов проектирования архитектуры и готовый шаблон для собственного проекта.
