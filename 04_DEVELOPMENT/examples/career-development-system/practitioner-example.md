## Career Development System: Пример для практика

### Задача
Сформировать карьерный план развития для специалиста по данным.

### Действие
1. Запустить веб-приложение:
   ```bash
   cd components/career-development-system
   python src/api/app.py
   ```
2. Открыть в браузере `http://localhost:8000`
3. Заполнить профиль: текущие навыки, цели, временной горизонт
4. Получить персонализированный план в `reports/career_plan.md`

### Результат
Персонализированный карьерный план с конкретными шагами и рекомендациями.
