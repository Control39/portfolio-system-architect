# Примеры использования компонентов

> Практические примеры для разных категорий пользователей

## Структура примеров

Для каждого компонента представлены примеры для трёх аудиторий:

- **Разработчик** — как модифицировать и расширять компонент
- **Практик** — как применять компонент для решения рабочих задач
- **Новичок** — как использовать компонент для обучения и карьерного роста
