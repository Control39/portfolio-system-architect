# Диаграммы системного мышления

## Системная карта

```mermaid
graph TB
    A[Внешняя среда] --> B[Система]
    B --> C[Компонент 1]
    B --> D[Компонент 2]
    B --> E[Компонент 3]
    C --> F[Подкомпонент 1.1]
    C --> G[Подкомпонент 1.2]
    D --> H[Подкомпонент 2.1]
    D --> I[Подкомпонент 2.2]
    E --> J[Подкомпонент 3.1]
    E --> K[Подкомпонент 3.2]

    F --> L[Элемент 1.1.1]
    F --> M[Элемент 1.1.2]
    G --> N[Элемент 1.2.1]
    G --> O[Элемент 1.2.2]

    H --> P[Элемент 2.1.1]
    H --> Q[Элемент 2.1.2]
    I --> R[Элемент 2.2.1]
    I --> S[Элемент 2.2.2]

    J --> T[Элемент 3.1.1]
    J --> U[Элемент 3.1.2]
    K --> V[Элемент 3.2.1]
    K --> W[Элемент 3.2.2]

    style B fill:#f9f,stroke:#333,stroke-width:4px
    style A fill:#bbf,stroke:#333,stroke-width:2px
```

## Причинно-следственная диаграмма

```mermaid
graph LR
    A[Изменение внешней среды] --> B[Влияние на систему]
    B --> C[Изменение состояния компонента 1]
    B --> D[Изменение состояния компонента 2]
    C --> E[Обратная связь на систему]
    D --> F[Обратная связь на систему]
    E --> G[Изменение внешней среды]
    F --> G
    G --> H[Новое состояние системы]

    style A fill:#f9f,stroke:#333
    style H fill:#bbf,stroke:#333
