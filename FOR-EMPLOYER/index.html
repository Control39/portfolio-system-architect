    Кейс 1: Arch-Compass (Infrastructure Orchestration Framework)

    Проблема: Управление сложной когнитивной архитектурой требует единого оркестратора для PowerShell, AI-провайдеров и DevOps процессов.

    Решение: Спроектировала и реализовала Arch-Compass Framework — контейнеризованный оркестратор на PowerShell с архитектурными контрактами и self-audit.

    Результат: - Единый интерфейс для управления AI (OpenAI, YandexGPT, LocalLLM) - Self-audit и экспорт метрик в Prometheus для production-ready системы - Доказанная безопасность: gitleaks, маскирование секретов, валидация конфигураций - Технологии: PowerShell 7, Docker, gitleaks, Pester, Prometheus

    Файлы: apps/infra-orchestrator/, README.md

    Кейс 2: MCP Server (AI Integration Platform)

    Проблема: AI-ассистенты (Claude, Koda) не имели доступа к коду и данным проекта.

    Решение: Спроектировала и реализовала MCP Server с 7 группами инструментов: - File Tools, Git Tools, IT-Compass Tools, ChromaDB Tools, Monitoring Tools, App Management, Security Tools

    Результат: - AI может читать/писать файлы, запускать сканирование безопасности, делать векторный поиск - Интеграция с Claude Desktop, Koda, VS Code - Технологии: Python, FastAPI, ChromaDB, FastMCP

    Файлы: apps/mcp-server/, docs/mcp-server/QUICKSTART.md

    Кейс 3: IT-Compass (Competency Framework)

    Проблема: Новички в IT не понимают как оценить свои компетенции. HR не могут объективно оценить кандидатов.

    Решение: Создала методологию с объективными маркерами по 19 доменам (Python, Docker, DevOps, System Thinking и др.)

    Результат: - 19 доменов, 5 уровней в каждом - Автоматическое обнаружение маркеров из кода - Генерация портфолио на основе маркеров - Технологии: Python, JSON, RAG для поиска маркеров

    Файлы: apps/it-compass/, docs/methodology/
