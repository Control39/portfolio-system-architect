# .kodacli/

Конфигурации для Koda CLI (командной строки).

## Назначение
- Хранение конфигураций и состояния командной строки Koda
- Управление настройками CLI-инструмента
- Хранение кэша и временных файлов CLI

Этот каталог содержит данные, специфичные для работы с Koda через командную строку, что позволяет автоматизировать задачи и интегрировать Koda в скрипты и рабочие процессы.
