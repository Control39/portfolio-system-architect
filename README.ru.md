## 🏗️ Архитектурное видение

Этот портфолио демонстрирует **когнитивную архитектуру** — новый подход к системному дизайну, который объединяет:

1. **Человеческое стратегическое мышление** с **AI-исполнительными возможностями**
2. **Production-ready паттерны** с первого дня (Kubernetes, мониторинг, безопасность)
3. **Воспроизводимую методологию** для перехода от non-technical к архитектору
4. **Enterprise-фокус** с конкретной релевантностью для российского рынка

### Ключевая инновация: От нуля до архитектора

Экосистема предоставляет полный путь для людей без IT-бэкграунда, чтобы стать enterprise-архитекторами через:
- **IT-Compass**: Объективное измерение компетенций
- **Cloud-Reason**: AI-ассистированное системное мышление
- **Portfolio-Organizer**: Автоматический сбор доказательств
- **Production deployment**: Симуляция реального опыта

## 🎯 IT-Compass: Объективное измерение компетенций

Рождённый из личного опыта перехода от нулевых IT-знаний к enterprise-архитектуре, IT-Compass предоставляет структурированную систему для измерения технических компетенций с объективными, проверяемыми маркерами.

**Ключевые особенности:**
- **18 областей навыков** (DevOps, Python, Системное мышление, Cloud Computing и др.) с тремя уровнями владения (начальный, средний, продвинутый)
- **Маркеры на основе SMART-критериев** — каждый маркер Specific, Measurable, Achievable, Relevant и Time-bound
- **Методология Екатерины Кудели** под лицензией CC BY-ND 4.0
- **Интеграция с Portfolio-Organizer** для автоматического сбора доказательств
- **CareerTracker** Python класс для мониторинга прогресса и рекомендаций

**Недавнее улучшение:** Все маркеры навыков стандартизированы с последовательной трёхуровневой структурой, заменяя общие маркеры конкретными, измеримыми целями. Система теперь включает 19 файлов навыков с 1495 маркерами на всех уровнях владения.

## 🚀 Быстрый старт

### Для HR и рекрутеров
1. **Изучите** [ONE-PAGE-SUMMARY](docs/employer/ONE-PAGE-SUMMARY.md) — краткое резюме проекта
2. **Посмотрите** [демо IT-Compass](apps/it-compass/README.md) — как работает измерение компетенций
3. **Оцените** [бизнес-ценность](docs/FOR-RUSSIAN-ENTERPRISE.md) для российских компаний

### Для разработчиков
```bash
# Клонируйте репозиторий
git clone https://github.com/Control39/cognitive-systems-architecture.git
cd cognitive-systems-architecture

# Настройте среду разработки (Windows)
.\scripts\dev\setup-dev.ps1

# Запустите IT-Compass
cd apps\it-compass
python src\main.py
```
