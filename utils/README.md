# utils/

Вспомогательные утилиты и скрипты.

## Назначение
- Хранение общих вспомогательных скриптов и утилит
- Обеспечение повторного использования кода для рутинных задач
- Поддержка процессов настройки и обслуживания

Этот каталог содержит различные утилиты, которые помогают в повседневной работе с проектом, от настройки окружения до выполнения специфических задач, которые не относятся к основной логике приложений.
