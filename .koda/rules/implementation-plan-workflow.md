## 🚫 ВАЖНЫЕ ПРАВИЛА

1. **НЕ выполнять задачи без подтверждения пользователя** (кроме критичных security)
2. **ВСЕГДА читать план** перед началом работы
3. **ВСЕГДА обновлять план** после выполнения
4. **ВСЕГДА делать коммит** с понятным сообщением
5. **НЕ пропускать задачи** — выполнять по порядку приоритета
6. **ЕСЛИ задача не может быть выполнена** — объяснить почему и предложить альтернативу
7. **ЕСЛИ пользователь просит пропустить** — отметить как [!] Пропущено с причиной
