# .koda/

Конфигурации для Koda - системы ИИ-агентов.

## Назначение
- Хранение правил, навыков и профилей для системы Koda
- Управление поведением ИИ-агентов в рамках экосистемы Koda
- Обеспечение согласованности между различными агентами

## Структура
- `rules/` - правила поведения агентов
- `skills/` - навыки и компетенции агентов (анализ, аудит, профилирование)
- `profiles/` - профили пользователей и агентов

Этот каталог интегрируется с системой Koda, обеспечивая продвинутые функции анализа кода, аудита безопасности и оптимизации процессов разработки.
