# migrations/

Миграции базы данных.

## Назначение
- Хранение скриптов для изменения схемы базы данных
- Обеспечение версионности и воспроизводимости изменений БД
- Управление эволюцией структуры данных

Миграции позволяют вносить изменения в структуру базы данных контролируемым образом, обеспечивая, что все участники проекта и окружения (разработка, тестирование, production) используют одинаковую версию схемы БД. Каждая миграция представляет собой скрипт, который может применить или откатить конкретное изменение.
