## 📞 Как использовать агент теперь

### Быстрый старт
```bash
cd portfolio-system-architect

# Проверить статус
python apps/cognitive-agent/launch-script.py --status

# Запустить в режиме scanning
python apps/cognitive-agent/scripts/scanner_main.py

# Запустить полный агент
python apps/cognitive-agent/launch-script.py --mode full
```

### Просмотр логов
```bash
# Последние 50 строк логов сканера
Get-Content apps/cognitive-agent/logs/scanner.log -Tail 50

# Последние 50 строк логов запуска
Get-Content apps/cognitive-agent/logs/agent_launch.log -Tail 50
```

### Просмотр отчётов
```bash
# Список всех отчётов
Get-ChildItem apps/cognitive-agent/reports/scans/

# Содержимое последнего отчёта
Get-Content apps/cognitive-agent/reports/scans/scan_report_*.json | ConvertFrom-Json
```
