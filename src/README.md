# src/

Исходный код основных компонентов системы.

## Назначение
- Хранение основного исходного кода проекта
- Централизованное расположение общих библиотек и утилит
- Обеспечение повторного использования кода между приложениями

## Структура
- `ai/` - компоненты, связанные с искусственным интеллектом
- `assistant_orchestrator/` - оркестратор ИИ-ассистентов
- `core/` - базовые компоненты и утилиты
- `decision_engine/` - движок принятия решений
- `embedding_agent/` - агент эмбеддингов
- `infrastructure/` - компоненты управления инфраструктурой
- `queues/` - управление очередями и асинхронными задачами
- `repo_audit/` - инструменты аудита репозитория
- `security/` - компоненты безопасности
- `shared/` - общие библиотеки и схемы

Этот каталог содержит общий код, который может использоваться несколькими приложениями из папки `apps/`, обеспечивая согласованность и избегая дублирования логики.
