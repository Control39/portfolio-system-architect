# Конфигурационные файлы

> Централизованное хранение конфигураций для всех компонентов

## Назначение

Папка `config/` содержит общие конфигурационные файлы для проекта:

- `project-config.yaml` — основные настройки проекта
- `component-config.yaml` — конфигурация компонентов
- `code-quality.yaml` — настройки качества кода

## Использование

Конфигурации компонентов находятся в соответствующих папках:

- `components/it-compass/config/`
- `components/cloud_reason/configs/`
- `components/ml-model-registry/config/`
