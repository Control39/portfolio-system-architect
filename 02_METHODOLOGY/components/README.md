# Компоненты Portfolio System Architect

Эта папка содержит упрощенные версии ключевых компонентов из ранее проанализированных проектов:

1. **it-compass** - система отслеживания компетенций
2. **arch-compass-framework** - архитектурный фреймворк
3. **cloud-reason** - система анализа облачных решений
4. **portfolio-organizer** - организатор портфолио
5. **system-proof** - исследования системной архитектуры
6. **thought-architecture** - портфолио когнитектурных решений
7. **career-development-system** - система карьерного развития
8. **ml-model-registry** - реестр ML-моделей с тестированием и версионированием

Каждый компонент содержит ключевые файлы и документацию, демонстрирующие основные функции соответствующего проекта.
