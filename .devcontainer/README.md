# .devcontainer/

Конфигурации для Dev Containers (контейнеров разработки).

## Назначение
- Определение стандартизированной среды разработки в Docker-контейнере
- Обеспечение одинаковой среды для всех разработчиков
- Упрощение настройки рабочего окружения

## Структура
- Содержит конфигурационные файлы Docker и VS Code для создания контейнера разработки

Этот каталог позволяет запускать полную среду разработки в контейнере, что гарантирует согласованность между локальными установками и CI/CD.
