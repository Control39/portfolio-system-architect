# Система управления карьерным развитием

## Описание

Система управления карьерным развитием на основе объективных маркеров компетенций. Позволяет отслеживать навыки пользователей, устанавливать цели развития и оценивать прогресс.

## Архитектура

### Компоненты

1. **API (Flask)**
   - Основной backend сервис
   - SQLAlchemy ORM для работы с базой данных
   - REST API эндпоинты

2. **Веб-интерфейс**
   - HTML/JavaScript клиент
   - Взаимодействует с API через fetch

3. **Мобильное приложение**
   - React Native (псевдокод)
   - Для демонстрации возможностей мобильной платформы

4. **Ядро системы**
   - CompetencyTracker: основной класс для отслеживания компетенций
   - Модели SQLAlchemy для ORM

5. **Вспомогательные модули**
   - Функции для работы с JSON
   - Валидация email
   - Форматирование дат и размеров файлов

## API эндпоинты

### Пользователи
- `GET /api/users` - Получить список всех пользователей
- `POST /api/users` - Создать нового пользователя

### Навыки
- `GET /api/users/<int:user_id>/skills` - Получить навыки пользователя
- `POST /api/users/<int:user_id>/skills` - Добавить навык пользователю

### Маркеры компетенций
- `GET /api/markers` - Получить все маркеры компетенций

## Запуск

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Запустите API:
   ```
   python src/api/app.py
   ```

3. Откройте `src/web/index.html` в браузере
