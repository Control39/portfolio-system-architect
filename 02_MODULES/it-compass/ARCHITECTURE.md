# Архитектура проекта IT-Compass

## Обзор архитектуры

IT-Compass - это методология объективных маркеров компетенций, которая предоставляет систему отслеживания прогресса и интеграции с портфолио.

## Основные компоненты

### 1. Архитектурные решения
- **decisions/**: Директория с ADR (Architectural Decision Records)

### 2. Схемы архитектуры
- **architecture/**: Директория со схемами архитектуры

### 3. Шаблоны
- **templates/**: Директория с шаблонами документов

## Технологический стек

- Markdown для документации
- JSON для хранения данных о маркерах

## Интеграция с другими проектами

### Зависимости
- Нет прямых зависимостей от других проектов в экосистеме

### Точки интеграции
- Интеграция с Portfolio-Organizer для автоматической генерации материалов
- Интеграция с Cloud-Reason для анализа контекста и извлечения маркеров
- Использование данных из IT-Compass в Career Development System

## Развитие проекта

### Планы по развитию
- Добавление первого ADR
- Подключение к Cloud-Reason
