## 1. 🎯 Позиционирование сервиса

### Как сервис представляется в документации

**Cognitive Agent** позиционируется как "мозг" экосистемы, центральный оркестратор, который связывает методологию IT-Compass, reasoning Decision Engine и автоматизацию Job Automation в единую систему.

**История создания:**
- **Декабрь 2025:** Первая идея агента (первый "день рождения")
- **Январь 2026 - Апрель 2026:** ⚠️ Период сломанной системы (ИИ-агенты не поняли архитектуру)
- **23 мая 2026:** 🔄 Второе рождение — восстановление и сборка по крупицам
- **Май 2026:** MVP (Minimum Viable Product) — базовая работоспособность

**Порт:** 8008
**Маршрут:** /cognitive-agent
**Архитектор:** @Control39 (Екатерина Куделя)

### Роль в экосистеме проекта

Cognitive Agent — **центральный оркестратор** в архитектуре "Атомы и Молекулы", связывающий все сервисы:
- IT-Compass (методология)
- Decision Engine (reasoning)
- Job Automation (поиск работы)
- Knowledge Graph (знания)
- Portfolio Organizer (доказательства)
- Career Development (карьера)

### Ключевые задачи
1. **Сканирование** — анализ проектов и понимание их структуры
2. **Планирование** — создание планов задач через ИИ (LangChain + GigaChat/Ollama)
3. **Выполнение** — выполнение задач через навыки (skills)
4. **Обучение** — сбор метрик и адаптация на результатах
5. **Интеграция** — связывание с другими сервисами

## 3. 🔗 Интеграции с другими компонентами

### Внутренние интеграции
- **IT-Compass** — сканирование маркеров компетенций (🟡 в разработке)
- **Decision Engine** — AI Reasoning через API (✅ базовая)
- **Knowledge Graph** — векторный поиск через ChromaDB (🟡 восстанавливается)
- **Career Development** — отправка метрик прогресса (✅ готово)
- **Portfolio Organizer** — экспорт результатов (✅ готово)
- **AI Config Manager** — централизованное управление конфигами (✅ работает)

### Внешние интеграции
- **GigaChat** — AI провайдер (✅ основной)
- **Ollama** — fallback AI (🟡 в планах)
- **ChromaDB** — Vector DB (🟡 в планах)
- **GitHub/GitLab** — SCM (🟡 в планах)
- **Jira/Trello** — Project Mgmt (🟡 в планах)
- **Slack/Telegram** — Comms (🟡 в планах)
- **Prometheus** — Monitoring (🟡 в планах)

### Архитектура интеграций

**Принцип "Атомы и Молекулы":**
- **Атомы (src/):** Переиспользуемые компоненты
  - `src/ai/gigachat_bridge.py` — AI интеграция
  - `src/shared/models.py` — Pydantic схемы
  - `src/core/config_loader.py` — загрузчик конфигов
- **Молекулы (apps/cognitive_agent/):** Конкретный сервис
  - `scripts/` — бизнес-логика
  - `src/` — FastAPI сервер
  - `config/` — конфигурация
