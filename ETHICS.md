# Ответственное использование ИИ в этом проекте

## 🔍 Валидация выводов
- Все рекомендации ИИ проходят проверку через архитектурные контракты
- Критические решения требуют человеческого подтверждения
- Механизм валидации реализован в `src/ai_orchestration/validation.py`

## ⚠️ Ограничения
- Модель не обучалась на персональных данных
- Не используется для принятия кадровых/финансовых решений
- Генерация кода ограничена внутренними компонентами портфолио

## 🛡️ Безопасность
- Секреты изолированы через sealed-secrets
- Сканирование уязвимостей в CI (Trivy, Bandit)
- Все входные данные для ИИ валидируются

## 📜 Лицензия и использование
- Код открыт, но коммерческое использование требует согласования
- Генерация кода осуществляется в рамках MIT-лицензии исходных библиотек
- Использование ИИ-провайдеров (YandexGPT, OpenAI) соответствует их ToS
