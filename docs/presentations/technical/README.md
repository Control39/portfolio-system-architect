# Технические презентации

Эта папка предназначена для технических презентаций, архитектурных обзоров, deep‑dive материалов, документации по компонентам и другим техническим аспектам системы.

## Как добавить

1. Создайте подпапку с названием презентации (например, `ml-model-registry-architecture`).
2. Поместите туда файлы презентации (Markdown, PDF, PPTX, диаграммы).
3. Добавьте `README.md` с кратким описанием и ссылками.

## Примеры тем

- Архитектура ML‑Model‑Registry
- Система reasoning в Cloud‑Reason
- Интеграция компонентов портфолио
- Performance tuning и масштабирование
- Security review и best practices
