## Как добавить новую презентацию

1. Выберите соответствующую категорию.
2. Создайте подпапку с понятным именем (например, `workshop/2025-ai-workshop`).
3. Добавьте файлы презентации (Markdown, PDF, PPTX) и README с описанием.
4. Обновите этот файл, добавив ссылку на новую презентацию.

## Форматы

Предпочтительный формат — Markdown (`.md`), так как он легко версионируется и читается в GitHub. Допустимы также PDF, PowerPoint, Google Slides (ссылкой).
