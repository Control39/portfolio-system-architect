# 🎬 Demo: IT Compass + Job Automation Agent

## 🎯 Цель

Показать, как **ваша авторская методология IT Compass** работает на практике и решает проблему карьерной неопределённости.

**Важно:** Демо работает в 3 режимах:
- ✅ **Демо-режим** — работает сразу, показывает шаблоны
- ✅ **Habr Career** — реальный поиск без регистрации
- ✅ **hh.ru API** — после регистрации на dev.hh.ru (по желанию)

## 🏗️ Архитектура

```
IT Compass (CareerTracker)
    ↓ загружает маркеры из apps/it_compass/src/data/markers/
    ↓ сканирует прогресс (25/83 маркера)
    ↓
Job Automation Agent
    ↓ анализирует навыки
    ↓ ищет вакансии
    ↓
Результат: подходящие вакансии + план развития
```
