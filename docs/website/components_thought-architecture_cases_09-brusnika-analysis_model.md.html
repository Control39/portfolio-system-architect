# Системный анализ: причинно‑следственные связи

## Модель финансовой ловушки роста:

Агрессивный рост + Отток ликвидности + Внешние шоки → Дефолт

## Количественная оценка рисков:

### Вероятность дефолта (PD):

PD = EBITDA/Чистый долг × 1/ICR

Для «Брусники»:
Чистый долг / EBITDA = 6,9x;
ICR = 0,93.

Тогда:
PD ≈ 6,9 × 1/0,93 ≈ 7,42

Значение >5 указывает на высокий риск дефолта.

### Запас прочности (SP):

SP = Собственный капитал/Общие активы

Для «Брусники»: SP = 4 %. Норма: ≥10 %.
