# Практическое применение модели

## Алгоритм системного аудита финансовой устойчивости:

1. Рассчитайте PD (вероятность дефолта) по формуле выше.
2. Оцените запас прочности (SP). Норма: ≥10 %.
3. Проверьте ICR. Норма: ≥1,5.
4. Проанализируйте структуру капитала (доля собственного капитала).
5. Спрогнозируйте WACC при изменении ставок.
