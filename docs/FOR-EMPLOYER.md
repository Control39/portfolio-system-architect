## 🏆 3 кейса для собеседования

### Кейс 1: MCP Server (AI Integration Platform)

**Проблема:** AI-ассистенты (Claude, Koda) не имели доступа к коду и данным проекта.

**Решение:** Спроектировала и реализовала MCP Server с 7 группами инструментов:
- File Tools, Git Tools, IT-Compass Tools, ChromaDB Tools, Monitoring Tools, App Management, Security Tools

**Результат:**
- AI может читать/писать файлы, запускать сканирование безопасности, делать векторный поиск
- Интеграция с Claude Desktop, Koda, VS Code
- **Технологии:** Python, FastAPI, ChromaDB, FastMCP

**Файлы:** `apps/mcp-server/`, `docs/mcp-server/QUICKSTART.md`

---

### Кейс 2: IT-Compass (Competency Framework)

**Проблема:** Новички в IT не понимают как оценить свои компетенции. HR не могут объективно оценить кандидатов.

**Решение:** Создала методологию с 1495 объективными маркерами по 18 доменам (Python, Docker, DevOps, System Thinking и др.)

**Результат:**
- 18 доменов, 5 уровней в каждом
- Автоматическое обнаружение маркеров из кода
- Генерация портфолио на основе маркеров
- **Технологии:** Python, JSON, RAG для поиска маркеров

**Файлы:** `apps/it-compass/`, `docs/methodology/`

---

### Кейс 3: Экосистема из 12 микросервисов

**Проблема:** Нужна масштабируемая архитектура для AI-системы.

**Решение:** Спроектировала и реализовала 12 независимых сервисов:
- `auth-service` — аутентификация
- `cloud-reason` — reasoning-движок
- `knowledge-graph` — граф знаний
- `portfolio-organizer` — авто-генерация портфолио
- `mcp-server` — AI integration
- `it-compass` — методология
- и ещё 7 сервисов

**Результат:**
- Каждый сервис — независимый Docker-контейнер
- Общий API Gateway
- Мониторинг: Prometheus + Grafana
- CI/CD: GitHub Actions
- **Технологии:** Python, FastAPI, Docker, Kubernetes, PostgreSQL

**Файлы:** `apps/`, `deployment/k8s/`, `docker-compose.yml`
