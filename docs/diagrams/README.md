# diagrams/services/

Отдельные диаграммы (Mermaid .mmd) по каждому сервису/модулю.

## Как использовать
- Каждый файл диаграммы назван так: `service-<key>.mmd`
- В корневом `README.md`/`docs/README.md` есть ссылка на master-схему: `docs/diagrams/master-system.mmd`
- Локальный индекс всех сервисных диаграмм: `docs/diagrams/services/diagrams-index.mmd`
