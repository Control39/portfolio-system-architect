## 🚀 Как использовать:
```bash
# Демо всей экосистемы
docker compose up -d
open http://localhost:8501  # IT-Compass

# С мониторингом
docker compose -f docker-compose.yml -f docker-compose.monitoring.yml up -d
open http://localhost:3000  # Grafana (admin/admin)

# С MLflow
docker compose -f docker-compose.yml -f docker-compose.mlflow.yml up -d
open http://localhost:5000  # MLflow UI

# Запустить тесты
pytest apps/ --cov=apps/ --cov-report=term-missing

# Сгенерировать документацию
cd docs/api && make html && open _build/html/index.html
```
