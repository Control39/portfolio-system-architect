## 📅 Среднесрочные цели (1-2 месяца)

### 5. Поиск бизнес-ценности в сервисах
**Описание:**
Агент должен анализировать каждый сервис и определять:
- Какую бизнес-проблему решает
- Кто стейкхолдеры
- Какие метрики успеха
- Как можно комбинировать с другими сервисами

**Реализация:**
```python
# .agents/skills/business_value_analyzer.py

def analyze_business_value(service_path: Path) -> dict:
    """
    Анализ бизнес-ценности сервиса
    """
    # 1. Чтение README, кода, конфигов
    # 2. Определение проблематики
    # 3. Выявление стейкхолдеров
    # 4. Оценка ROI
    # 5. Предложения по комбинированию
    
    return {
        "problem": "Проблема, которую решает",
        "stakeholders": ["HR", "Разработчики", "Бизнес"],
        "roi": "Экономия X часов в месяц",
        "combinations": [
            {"with": "service_X", "benefit": "Новая фича Y"}
        ]
    }
```

**Статус:** ⚪ В планах

### 6. Комбинирование сервисов для архитектурных схем
**Описание:**
Агент анализирует все сервисы и создаёт:
- Диаграммы связей (Mermaid)
- Сценарии использования (use cases)
- Предложения по объединению в микросервисные кластеры

**Реализация:**
```python
# .agents/skills/architecture_composer.py

def compose_architecture(services: list) -> dict:
    """
    Создание архитектурной схемы из сервисов
    """
    # 1. Анализ зависимостей между сервисами
    # 2. Выявление паттернов (MVC, Event-Driven, и т.д.)
    # 3. Генерация Mermaid диаграммы
    # 4. Предложения по оптимизации
    
    return {
        "diagram": "mermaid код",
        "patterns": ["MVC", "CQRS"],
        "optimizations": ["Объединить A и B", "Разделить C"]
    }
```

**Статус:** ⚪ В планах
