🔄 Потенциальные зависимости между компонентами
Согласно архитектуре проекта, компоненты связаны следующим образом:

AI Provider Manager ← AI Config Manager (настройки провайдеров)
Cognitive Agent ← AI Provider Manager, IT Compass, Embedding Agent
GigaChat Bridge ← AI Provider Manager (потенциально)
Project Scanner ← AI Config Manager (для получения конфигурации сканирования)
