# Якорь для IT-рекрутеров

Этот анализ демонстрирует навык, критичный для Product Owner / Business Analyst / System Architect: умение выявлять структурные риски в сложных системах — будь то финансовая модель девелопера или архитектура микросервисов.

Анализ проведён с использованием маркеров системного мышления из методологии IT-Compass: [ссылка на методологию](https://github.com/leadarchitect-ai/portfolio-system-architect/tree/main/components/it-compass)
