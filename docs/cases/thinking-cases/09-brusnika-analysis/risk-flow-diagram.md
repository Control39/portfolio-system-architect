# Визуализация системы (упрощённая схема)

## Компоненты модели:

- Вход: агрессивный рост + отток ликвидности.
- Процесс: рост долга → увеличение процентных расходов → снижение ликвидности.
- Выход: дефолт или реструктуризация.
