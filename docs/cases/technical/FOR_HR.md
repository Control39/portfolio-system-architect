# AI Config Manager: Для HR и Hiring Managers

## Кратко

**Что это:** Десктоп-приложение для управления AI-сервисами.

**Зачем:** Разработчикам нужно быстро переключаться между разными AI (Ollama, GigaChat, YandexGPT), мониторить их статус и настраивать AI-агентов.

## Какие проблемы решает

❌ **До:**
- 5 разных интерфейсов для AI
- Ручное редактирование YAML
- Нет мониторинга

✅ **После:**
- 1 интерфейс для всех AI
- Конфиги генерируются автоматически
- Real-time статус всех сервисов

## Почему это впечатляет

1. **Full-stack + Desktop + Mobile** — редкое комбо навыков
2. **Интеграция с 5+ провайдерами** — каждый со своим API
3. **Real-time мониторинг** — WebSocket, не тривиальная задача
4. **Самостоятельный продукт** — не pet-project, а реальный инструмент

## Вопросы для интервью

**Q:** Почему вы выбрали Electron, а не веб-приложение?

**A:** Electron дает доступ к файловой системе для генерации конфигов и возможность работы без интернета с локальными моделями (Ollama).

**Q:** Как вы обеспечиваете безопасность API-ключей?

**A:** Ключи хранятся в .env файле (не в Git) и подставляются через переменные окружения. При генерации конфигов используются \ плейсхолдеры.

**Q:** Как тестировали интеграцию с разными API?

**A:** Создал моки для каждого провайдера и написал unit-тесты. Для e2e использовал реальные API с тестовыми ключами.
