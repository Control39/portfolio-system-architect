# Связь с IT‑Compass

*Этот файл будет заполнен в рамках развития проекта. Здесь будет описано, как система управления знаниями интегрируется с IT‑Compass и какие преимущества это даёт.*

## Концепция интеграции
IT‑Compass — это портфолио‑система, которая помогает IT‑специалистам структурировать свои достижения, планировать карьеру и получать психологическую поддержку. Система управления знаниями (Knowledge Management System, KMS) становится её ядром, обеспечивая:

1. **Структурирование компетенций** через маркеры.
2. **Трекинг прогресса** по каждому маркеру.
3. **Персонализированные рекомендации** материалов для изучения.
4. **Психологическую поддержку** на основе анализа прогресса и состояния пользователя.

## Техническая интеграция
### Обмен данными
- KMS использует те же JSON‑файлы маркеров, что и IT‑Compass.
- Прогресс пользователя синхронизируется между системами через общий `user_progress.json`.
- Модуль психологической поддержки KMS предоставляет API, которое может вызывать IT‑Compass для показа мотивационных сообщений.

### Архитектурные решения
- KMS работает как подмодуль IT‑Compass (или как отдельный микросервис).
- CLI‑интерфейс KMS доступен из командной строки IT‑Compass.
- Веб‑интерфейс IT‑Compass отображает прогресс из KMS через виджеты.

## Преимущества интеграции
- **Единая точка входа** для пользователя — не нужно переключаться между системами.
- **Обогащение данных** — прогресс по маркерам автоматически учитывается в портфолио.
- **Усиление психологической поддержки** — KMS анализирует глубинные паттерны обучения и даёт более точные рекомендации по предотвращению выгорания.
- **Масштабируемость** — новые маркеры и материалы добавляются один раз, становятся доступны в обеих системах.

## Дорожная карта интеграции
1. **Фаза 1** (текущая) — создание заглушек и структуры папок.
2. **Фаза 2** — рефакторинг core‑модулей и написание тестов.
3. **Фаза 3** — подключение KMS к IT‑Compass через API.
4. **Фаза 4** — пользовательское тестирование и сбор обратной связи.
5. **Фаза 5** — публичный релиз в рамках гранта SourceCraft.
