# Documentation Automation

Автоматизация генерации документации — исследование подходов к масштабируемому документированию проектов с использованием Python-скриптов.

## Описание

Этот кейс представляет исследование в области автоматизации документации, включая:
- Генерацию карты знаний для Obsidian
- Создание статического сайта-портфолио
- Автоматическое обновление через GitHub Actions

## Компоненты

### Scripts
- `generate_obsidian_map.py` — генератор карты знаний
- `generate_website.py` — генератор сайта-портфолио
- `run_daily.ps1` — ежедневная автоматизация

### Результаты
- Автоматическая генерация 300+ файлов документации
- Статический сайт без зависимостей Node.js
- Интеграция с GitHub Actions
