# Прототип системы управления знаниями

*Этот файл будет заполнен в рамках развития проекта. Здесь будет описание прототипа, его компонентов и результатов тестирования.*

## Цель прототипа
Проверить гипотезу о том, что маркеры компетенций могут быть эффективно использованы для структурирования знаний и рекомендаций.

## Компоненты прототипа
1. **База маркеров** — JSON‑файлы с описанием навыков.
2. **Трекер прогресса** — модуль для отслеживания изученных маркеров.
3. **Генератор рекомендаций** — алгоритм подбора материалов на основе прогресса.
4. **Психологическая поддержка** — модуль мотивации и анти‑выгорания.

## Реализация
- Использован Python для backend‑логики.
- Данные хранятся в JSON для простоты.
- Интерфейс командной строки (CLI) для взаимодействия.
