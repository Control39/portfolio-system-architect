# Материалы для воркшопов

Эта папка содержит материалы для проведения воркшопов, практических сессий, обучающих мероприятий и hands‑on tutorials.

## Структура

Рекомендуется создавать подпапку для каждого воркшопа с именем в формате:

```
YYYY‑MM‑DD‑topic‑workshop
```

Например:
- `2025‑03‑10‑system‑thinking‑workshop`
- `2025‑04‑15‑ml‑pipeline‑hands‑on`

Внутри каждой подпапки размещайте:
- `README.md` — описание воркшопа, agenda, цели, предварительные требования.
- Презентационные слайды (PDF, PPTX, Markdown).
- Практические задания (`.ipynb`, `.py`, `.md` с инструкциями).
- Ресурсы (датасеты, конфиги, шаблоны).
- Фотографии/видео (если есть).

## Как добавить новый воркшоп

1. Создайте подпапку с понятным именем.
2. Заполните её материалами.
3. Обновите этот README, добавив ссылку на новый воркшоп.

## Примеры тем

- System Thinking для архитекторов
- Построение ML‑Model‑Registry с нуля
- Интеграция IT‑Compass и Portfolio‑Organizer
- Reasoning loops и когнитивная архитектура
- Воркшоп по визуализации архитектуры (Mermaid, Draw.io)
