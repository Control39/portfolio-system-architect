# План интеграции проектов

## Цель
Объединить проекты Decision Engine, Infra-Orchestrator-Framework и другие компоненты портфолио в единую экосистему с shared reasoning API и автоматизацией.

## Текущее состояние

### Cloud-Reason
- ✅ Имеет REST API с эндпоинтами `/api/v1/reason`, `/api/v1/workflow`, `/api/v1/status`
- ✅ Интеграция с Git, сканирование секретов
- ✅ Документация в component-config.yaml
- ❌ Не хватает интеграции с YandexGPT API
- ❌ Нет загрузки файлов в Object Storage
- ❌ Нет веб-интерфейса для тестирования

### Infra-Orchestrator-Framework
- ✅ PowerShell фреймворк с модулями безопасности
- ✅ Интеграция с gitleaks, SecretManager
- ✅ Конфигурация через YAML
- ❌ Нет модуля для работы с reasoning API
- ❌ Нет генерации архитектурных диаграмм
- ❌ Нет поддержки MCP

### Другие проекты
- IT-Compass: трекинг навыков
- Portfolio-Organizer: веб-интерфейс
- System-Proof: верификация
- ML Model Registry: управление моделями

## Задачи интеграции

### 1. Расширение Decision Engine (Этап 1)
- [ ] Добавить эндпоинт `/chat` для диалога с AI (YandexGPT/GigaChat)
- [ ] Реализовать загрузку файлов через `/upload` в Object Storage
- [ ] Интегрировать ChromaDB для RAG (Retrieval Augmented Generation)
- [ ] Создать веб-интерфейс на FastAPI Swagger
- [ ] Настроить health checks и мониторинг

### 2. Модуль интеграции для Arch-Compass (Этап 2)
- [ ] Создать PowerShell модуль `CloudReason.psm1` с функциями:
  - `Invoke-CloudReasonAnalysis`
  - `Get-CloudReasonStatus`
  - `Send-FileToCloudReason`
- [ ] Добавить команду в ArchCompass: `Start-CloudReasonIntegration`
- [ ] Написать тесты Pester для модуля

### 3. Унификация документации (Этап 3)
- [ ] Обновить README decision-engine с примерами использования API
- [ ] Добавить раздел "Интеграция с Infra-Orchestrator" в документацию
- [ ] Создать общую диаграмму архитектуры (уже создана)
- [ ] Опубликовать документацию на GitHub Pages через MkDocs

### 4. Автоматизация CI/CD (Этап 4)
- [ ] Настроить workflow в SourceCraft для тестирования интеграций
- [ ] Добавить stage "integration-tests" в pipeline
- [ ] Реализовать деплой облачной версии Cloud-Reason на Yandex Cloud Functions
- [ ] Настроить автоматическое обновление документации

### 5. Дополнительные улучшения (Этап 5)
- [ ] Добавить поддержку MCP (Model Context Protocol) для Decision Engine
- [ ] Реализовать генерацию архитектурных диаграмм через Infra-Orchestrator (Mermaid)
- [ ] Интегрировать System-Proof для верификации reasoning результатов
- [ ] Создать демо-видео с полным workflow

## Ожидаемые результаты

1. **Decision Engine** станет полноценным reasoning API с возможностью:
   - Диалога с AI по контексту файлов
   - Автоматического анализа архитектурных решений
   - Интеграции с внешними инструментами через REST

2. **Arch-Compass-Framework** получит новые возможности:
   - Вызов reasoning API для анализа конфигураций
   - Генерация диаграмм на основе YAML
   - Улучшенная безопасность через интеграцию с Cloud-Reason

3. **Портфолио в целом** будет демонстрировать:
   - Глубокую интеграцию между проектами
   - Использование современных технологий (RAG, MCP, serverless)
   - Профессиональный подход к документации и автоматизации

## Критерии успеха

- [ ] Decision Engine API отвечает на запросы `/chat` с контекстом файлов
- [ ] Infra-Orchestrator может вызывать Decision Engine через PowerShell модуль
- [ ] Интеграционные тесты проходят в CI/CD
- [ ] Документация обновлена и доступна онлайн
- [ ] Демо-видео опубликовано

## Риски и митигации

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Задержки с интеграцией YandexGPT API | Средняя | Высокое | Использовать локальную модель (Llama) как fallback |
| Сложности с Object Storage | Низкая | Среднее | Использовать локальное хранилище для MVP |
| Нехватка времени на документацию | Высокая | Среднее | Привлечь технического писателя на part-time |

## Заключение

Интеграция проектов позволит создать целостную экосистему, демонстрирующую навыки архитектурного мышления, разработки и автоматизации. План реалистичен и может быть выполнен в течение 2-3 недель.
