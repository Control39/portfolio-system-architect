# Объединённая грантовая заявка SourceCraft

Этот грант объединяет:
- `01_proposal.md` - основная заявка с информацией о кандидате и проекте
- `02_cases.md` - три кейса с количественными метриками и экономическими расчётами

## Как использовать
1. Для грантового комитета - использовать оба документа вместе
2. Для презентации - брать кейсы из 02_cases.md
3. Для личной информации - данные из 01_proposal.md
