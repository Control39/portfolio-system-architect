# Дизайн скрипта для поиска дубликатов файлов

## Общее описание
Скрипт на Python для рекурсивного обхода директорий проекта с целью поиска дублирующихся файлов по хешу содержимого.

## Технические требования
- Язык: Python 3.x
- Использовать hashlib для вычисления хешей
- Игнорировать системные директории (.git, __pycache__, .vscode, .idea)
- Пропускать файлы нулевой длины
- Вывод результатов в виде таблицы с названием, содержимым и путем файла

## Архитектура решения

### 1. Рекурсивный обход директорий
- Использовать os.walk() для обхода всех поддиректорий
- Фильтровать игнорируемые директории
- Пропускать файлы нулевой длины

### 2. Вычисление хешей
- Использовать hashlib.sha256() для вычисления хеша содержимого файла
- Читать файлы блоками для эффективного использования памяти

### 3. Отслеживание дубликатов
- Хранить хеши в словаре {hash: [file_paths]}
- Идентифицировать дубликаты как хеши с несколькими путями

### 4. Формат вывода
Таблица с колонками:
- Название файла
- Содержимое (первые 100 символов)
- Полный путь к файлу

## План реализации
1. Создать функцию обхода директорий с фильтрацией
2. Реализовать вычисление хешей файлов
3. Создать структуру для хранения информации о файлах
4. Разработать функцию вывода результатов в виде таблицы
5. Добавить обработку ошибок и валидацию входных данных

## Игнорируемые директории
- .git
- __pycache__
- .vscode
- .idea
- node_modules (если присутствует)
