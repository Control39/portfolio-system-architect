# Как внести вклад в Манифест

> **Ваш вклад- это кирпичик в здании новой профессиональной реальности.**

---

## Откройте issue

### Если у вас есть идея
- Предложите новый элемент для документального фильма
- Поделитесь своим опытом, который может стать частью манифеста

### Если вы нашли проблему
- Опишите что не работает или непонятно
- Предложите исправление
- Укажите как это влияет на понимание новой роли

## Сделайте Pull Request

### Типы вклада
- **Контент**: Дополнения к историям, улучшения формулировок, переводы
- **Документация**: Новые разделы, улучшения структуры, визуализации
- **Код**: Улучшения скриптов генерации, исправление багов

### Процесс
1. Сделайте fork репозитория
2. Создайте отдельную ветку для вашего вклада
3. Внесите изменения и зафиксируйте их
4. Откройте Pull Request с описанием изменений

## Принципы вклада

1. **Подлинность**: Делитесь искренним опытом
2. **Уважение**: Цените путь каждого
3. **Сотрудничество**: Стройте вместе, а не против
4. **Эволюция**: Вклад- это не фиксация, а продолжение повествования.

 Ваш голос важен.
