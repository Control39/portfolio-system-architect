# История развития системы

> Хронология ключевых изменений и достижений

v1.0.0 - Начало пути (лето 2025)

- Создана первая версия IT-Compass как Excel-таблица для отслеживания навыков
- Начало формирования методологии объективных маркеров
- Поиск ответа на вопрос: "Кто я в IT, если я не кодер?"
- Первый репозиторий IT-Compass: 2025

## v1.5.0 - Систематизация (осень 2025)

- Переход от Excel к Python-реализации IT-Compass
- Создание Arch-Compass - архитектурного фреймворка
- Начало работы над RAG-системой для извлечения достижений

## v2.0.0 - Автоматизация (2025-2026)

- Полноценная RAG-система с интеграцией в LLM
- Создание ML Model Registry с 11 видами тестов
- Автоматическая генерация сайта и карты знаний
- 7 ADR-документов с архитектурными решениями

## v2.5.0 - Интеграция (2026)

- Объединение компонентов в единую экосистему
- Career Development System с мобильным и веб-приложением
- 8 аналитических кейсов (Uber Freight, Брусника, Patreon/Apple)
- Подача заявки на грант SourceCraft Open Source

## v3.0.0 - Реструктуризация (2026)

- Переход от нарративной к модульной структуре
- Создание единой системы организации репозитория
- Формализация роли "архитектор когнитивных систем"
