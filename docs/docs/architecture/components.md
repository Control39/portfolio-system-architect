# Компоненты системы

## it-compass

Методология объективных маркеров компетенций. Предоставляет:
- Систему отслеживания прогресса
- Генерацию портфолио на основе маркеров
- Интерфейсы для интеграции с другими компонентами

### Архитектурные особенности
- Модульная структура с четко определенными интерфейсами
- Поддержка различных форматов данных для маркеров
- Расширяемая система отслеживания прогресса

## arch-compass-framework

PowerShell фреймворк для архитектурных решений. Предоставляет:
- Инструменты для автоматизации архитектурных процессов
- Шаблоны для проектирования решений
- Интерфейсы для интеграции с системой управления проектами

### Архитектурные особенности
- Модульная архитектура с возможностью подключения плагинов
- Поддержка различных форматов конфигурации
- Интеграция с системами контроля версий

## portfolio-organizer

Инструмент для организации и представления портфолио. Предоставляет:
- Структурированное представление портфолио
- Инструменты для визуализации достижений
- Интерфейсы для интеграции с it-compass

### Архитектурные особенности
- Поддержка различных форматов представления данных
- Интеграция с системами визуализации
- Расширяемая система шаблонов

## cloud-reason

Исследование облачных решений и интеграции с YandexGPT. Предоставляет:
- Анализ облачных решений
- Интеграцию с ИИ для архитектурного анализа
- Инструменты для автоматизации принятия решений

### Архитектурные особенности
- Модульная архитектура для поддержки различных провайдеров
- Интеграция с ИИ для анализа решений
- Поддержка автоматизированного масштабирования

## system-proof

Исследования в области системной архитектуры. Предоставляет:
- Теоретическую базу для системного мышления
- Методологии для анализа архитектурных решений
- Кейсы и примеры успешных интеграций

### Архитектурные особенности
- Системный подход к анализу архитектурных решений
- Поддержка различных методологий анализа
- Интеграция с инструментами моделирования

## thought-architecture

Портфолио когнитивной архитектуры. Предоставляет:
- Примеры когнитивных моделей
- Методологии для проектирования мышления
- Инструменты для визуализации архитектуры мысли

### Архитектурные особенности
- Поддержка различных моделей когнитивной архитектуры
- Интеграция с инструментами визуализации
- Расширяемая система шаблонов для проектирования мышления

## career-development-system

Система карьерного развития с мобильным и веб-интерфейсами. Предоставляет:
- Трекинг компетенций и достижений
- Интеграцию с it-compass для объективных маркеров
- API для сторонних приложений

### Архитектурные особенности
- Модульная архитектура с разделением на backend и frontend
- Поддержка REST API и GraphQL
- Интеграция с системами аутентификации

## ml-model-registry

Реестр ML-моделей с тестированием и версионированием. Предоставляет:
- Хранение метаданных моделей и артефактов
- Автоматическое тестирование (контракты, fuzz, безопасность, производительность)
- Управление версиями и развертыванием

### Архитектурные особенности
- Модульная архитектура с плагинами для разных бэкендов хранения
- Поддержка 11 видов тестов (контракты, edge‑cases, fuzz, безопасность, интеграция, производительность, устойчивость)
- REST API и CLI для управления
