## ❓ Общие вопросы

### Что такое «когнитивный архитектор»?

**Когнитивный архитектор** — это специалист, проектирующий системы мышления и автоматизации с использованием ИИ. Роль сочетает:
- Технические навыки (архитектура, разработка, ML)
- Метакогнитивные навыки (системное мышление, рефлексия)
- Навыки коммуникации с ИИ (промпт-инжиниринг, RAG)

**См. также:** [about-me.md](../about-me.md), [docs/glossary.md](glossary.md)

### Чем отличается этот проект от обычного портфолио?

Обычное портфолио — это набор работ. Этот проект — **система демонстрации компетенций**, которая:
- Показывает не только результаты, но и методологию их достижения
- Включает работающие компоненты, а не только описания
- Автоматически генерирует документацию и визуализации
- Доказывает способность строить сложные системы (8 компонентов)

### Для кого этот проект?

- **Для работодателей:** демонстрация системного мышления и навыков архитектора
- **Для контрибьюторов:** готовые шаблоны и методология
- **Для изучения:** примеры использования ИИ в реальных задачах

## 🚀 Быстрый старт

### Как начать использовать проект?

```bash
# Клонировать репозиторий
git clone https://github.com/leadarchitect-ai/portfolio-system-architect.git
cd portfolio-system-architect

# Сгенерировать карту знаний для Obsidian
python scripts/generate_obsidian_map.py

# Сгенерировать сайт-портфолио
python scripts/generate_website.py
```

**См. также:** [README.md](../README.md#быстрый-старт)

### Как начать использовать IT-Compass?

1. Перейдите в `components/it-compass/`
2. Изучите структуру маркеров в `src/data/markers.json`
3. Запустите трекер: `python src/core/tracker.py`
4. Добавьте свои достижения в `src/data/user_progress.json`

**См. также:** [components/it-compass/ARCHITECTURE.md](../components/it-compass/ARCHITECTURE.md)

## 🧩 Компоненты

### Какие компоненты можно использовать независимо?

Все 8 компонентов спроектированы как независимые модули:

| Компонент | Независимость | Сложность |
|-----------|---------------|-----------|
| IT-Compass | ✅ Полная | Низкая |
| Portfolio Organizer | ✅ Полная | Низкая |
| Cloud Reason (RAG) | ✅ Полная | Средняя |
| ML Model Registry | ✅ Полная | Средняя |
| Career Development System | ✅ Полная | Средняя |
| Arch-Compass Framework | ✅ Полная | Высокая |
| System Proof | ⚠️ Требует RAG | Высокая |
| Thought Architecture | ⚠️ Требует RAG | Высокая |

### Какие технологии используются?

**Языки и фреймворки:**
- Python (основной)
- JavaScript/TypeScript (веб-компоненты)
- PowerShell (Windows-скрипты)

**ИИ и ML:**
- YandexGPT / OpenAI API

- Векторные базы данных

**Инфраструктура:**
- GitHub Actions (CI/CD)
- Docker
- Obsidian (для карты знаний)

## 🔧 Технические вопросы

### Почему используется несколько языков?

Проект демонстрирует polyglot-подход:
- **Python** — основной для ИИ/ML
- **JavaScript** — для веб-интерфейсов
- **PowerShell** — для Windows-автоматизации

Это показывает универсальность архитектора.

### Как часто обновляются компоненты?

- **Ядро (RAG, ML Registry):** ежемесячно
- **Документация:** по мере необходимости
- **Кейсы:** при появлении новых примеров
