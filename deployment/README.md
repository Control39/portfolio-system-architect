# deployment/

Конфигурации развертывания и инфраструктуры.

## Назначение
- Хранение конфигураций для развертывания приложений
- Управление инфраструктурой как кодом (IaC)
- Автоматизация процессов деплоя и управления окружениями

## Структура
- `gitops/` - конфигурации GitOps для управления состоянием кластера
- `k8s/` - манифесты Kubernetes для оркестрации контейнеров
- `secrets/` - конфигурации для управления секретами (Sealed Secrets)

Этот каталог содержит все необходимое для развертывания системы в production, включая оркестрацию контейнеров, управление конфигурациями и безопасное хранение секретов.
