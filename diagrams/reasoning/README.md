# Reasoning-графы

Эта директория содержит визуализации reasoning-графов для портфолио системного архитектора.

## Структура

Базовая структура reasoning-графа включает следующие узлы:

1. **Источник** - начальная точка, откуда берется задача или идея
2. **Задача** - конкретная задача, которую необходимо решить
3. **Инструмент** - инструменты и технологии, используемые для решения
4. **Направление** - направление развития (DevOps, MLOps, ИИ и т.д.)
5. **Подтверждение** - проверка правильности решения
6. **Результат** - конечный результат и его интеграция

## Форматы

- `graph-schema.json` - JSON-схема базовой структуры графа
- `.excalidraw` - файлы для визуализации в Excalidraw

## Использование

Для создания нового reasoning-графа:
1. Скопируйте `graph-schema.json` как основу
2. Отредактируйте узлы и связи под конкретную задачу
3. Визуализируйте в Excalidraw, импортировав JSON
4. Сохраните как `.excalidraw` файл

Для шаблона заполнения см. `docs/reasoning/TEMPLATE.md`.
