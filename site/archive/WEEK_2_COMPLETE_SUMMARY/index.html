    Service Structure (Standardized)

    apps/<service>/
├── src/                    ← Application code
├── config/                 ← Configuration
├── tests/
│   ├── test_basic.py      ← Enhanced tests (15 tests)

├── README.md              ← Service overview + testing info

└── Dockerfile            ← Container config
