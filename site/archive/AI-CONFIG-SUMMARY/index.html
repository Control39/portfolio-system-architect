    Quick Start

    Run Repository Audit

    python .gigacode/tools/repo-audit.py

    Run Secrets Scan

    python .gigacode/tools/secrets-scanner.py

    Run Security Scan (Workflow)

    # Follow .gigacode/workflows/security-scan.md
pip-audit
bandit -r src/ -ll
trufflehog filesystem .
