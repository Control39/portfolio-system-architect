# Обзор системы

## Описание
Portfolio-system-architect — это автоматизированная система документации для портфолио системного архитектора.

## Ключевые компоненты
- **Автоматизация генерации**: Скрипты для Obsidian-карты и статического сайта.
- **Структура репозитория**: 01_ARCHITECTURE/, 02_MODULES/, 03_CASES/ и др.
- **Инструменты**: Python-скрипты, PowerShell, Git.

## Цели
- Поддерживать актуальную документацию.
- Генерировать Obsidian-ноуты и HTML-сайт автоматически.
