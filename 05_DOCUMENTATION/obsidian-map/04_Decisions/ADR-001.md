# ADR-001: Выбор стека

**Статус**: Принято
**Дата**: 2024-10-01

## Контекст
Нужен простой стек без зависимостей.

## Варианты
1. Node.js + Next.js → отклонено (зависимости)
2. Python + markdown → принято (легкий)

## Последствия
Статический HTML, быстрая генерация.
