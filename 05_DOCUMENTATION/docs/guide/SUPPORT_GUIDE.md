# 🧠 IT COMPASS: РУКОВОДСТВО ПО ПСИХОЛОГИЧЕСКОЙ ПОДДЕРЖКЕ
## 🎯 Цель раздела
Этот раздел создан для того, чтобы помочь пользователям IT Compass справляться с эмоциональными и психологическими аспектами обучения и профессионального развития в IT.
## 🚀 Как использовать раздел поддержки
### Быстрый доступ
```python
src/main.py --mental-support
```
### Основные функции:
* 💬 **Мотивационные цитаты** - ежедневная поддержка
* 🆘 **Кризисные номера помощи** - экстренная поддержка
* 🧘 **Режим 'низкой энергии'** - для периодов усталости
* 👥 **Руководство сообщества** - помощь в построении здоровых отношений
