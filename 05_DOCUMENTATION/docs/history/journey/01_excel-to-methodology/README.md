# IT-Compass: Как табличка стала методологией

Этот раздел- первая глава моего документального фильма о рождении новой профессии. Здесь рассказывается, как в 2022 году, не имея опыта в IT, я создала Excel-таблицу для самоопределения, которая позже эволюционировала в полноценную методологию IT-Compass.

Это не просто инструмент, это доказательство того, как системное мышление может быть преобразовано в измеримую, объективную методологию. Моя таблица- это первый артефакт новой формы экспертизы, где личный путь стал воспроизводимой системой для развития компетенций.
