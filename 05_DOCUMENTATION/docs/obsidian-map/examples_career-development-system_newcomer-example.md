## Career Development System: Пример для новичка

### Задача
Определить направление карьерного роста в IT.

### Действие
1. Открыть веб-приложение:
   ```bash
   cd components/career-development-system
   python src/api/app.py
   ```
2. Перейти по адресу `http://localhost:8000`
3. Пройти опрос: текущие навыки, интересы, цели
4. Получить рекомендации по направлениям

### Результат
Чёткое понимание возможных карьерных путей в IT и план развития.
