# GIGA IDE MCP Server

Этот сервер обеспечивает централизованное управление промптами и правилами для GIGA IDE.

## Описание

MCP (Model Control Protocol) сервер обрабатывает промпты от GIGA IDE, применяет к ним набор правил и возвращает обработанные результаты. Сервер построен на FastAPI и предоставляет RESTful API для интеграции.

## Основные функции

- Обработка промптов с применением правил
- Управление наборами правил
- Проверка работоспособности
- Логирование операций

## Установка и запуск

1. Создайте виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
```bash
# Windows
venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите сервер:
```bash
python main.py
```

Или с помощью uvicorn напрямую:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## API Endpoints

- `GET /` - Проверка состояния сервера
- `GET /health` - Проверка работоспособности
- `GET /rules` - Получение списка правил
- `POST /process-prompt` - Обработка промпта

## Конфигурация

Правила хранятся в файле `rules.json`. Вы можете модифицировать существующие правила или добавлять новые.

## Логирование

Логи сервера сохраняются в директории `logs/`. Файл лога ротируется ежедневно.

## Безопасность

Сервер слушает на всех интерфейсах (0.0.0.0) на порту 8000. В production-среде рекомендуется использовать reverse proxy (nginx) и HTTPS.
