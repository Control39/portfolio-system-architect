## 🚨 Устранение неполадок

### Частые проблемы и решения

1. **Pre-commit hooks не работают**
```bash
pre-commit install  # Переустановить hooks
pre-commit run --all-files  # Запустить вручную
```

2. **GitHub Actions failing**
- Проверьте логи в Actions tab
- Убедитесь в наличии secrets
- Проверьте корректность YAML

3. **Docker не запускается**
```bash
docker system prune -a  # Очистка Docker
docker-compose down && docker-compose up -d  # Перезапуск
```

4. **Проблемы с зависимостями**
```bash
rm -rf .venv  # Удалить виртуальное окружение
./scripts/setup-environment.sh  # Переустановить
```
