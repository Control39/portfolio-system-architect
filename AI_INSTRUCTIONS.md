## 🔧 ВОССТАНОВЛЕНИЕ ОШИБОК

### Если ИИ удалил важный файл:

```bash
# 1. Найти последний коммит с файлом
git log --all --full-history -- "<путь_к_файлу>"

# 2. Восстановить из коммита
git checkout <commit-hash> -- <путь_к_файлу>

# 3. Добавить в .gitkeep (защита от удаления)
touch <путь_к_файлу>/.gitkeep
```

### Если ИИ переместил важный файл:

```bash
# 1. Найти файл
git log --all --full-history --diff-filter=R -- "<путь_к_файлу>"

# 2. Вернуть на место
git checkout <commit-hash> -- <старый_путь>

# 3. Удалить из нового места (если нужно)
git rm <новый_путь>
```
