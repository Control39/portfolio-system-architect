# .sourcecraft/

Конфигурации для SourceCraft - системы управления ИИ-агентами.

## Назначение
- Хранение навыков и правил для системы SourceCraft
- Управление поведением агентов в рамках SourceCraft экосистемы
- Централизованное управление качеством кода и процессами разработки

## Структура
- `skills/` - навыки агентов для анализа и аудита кода

Этот каталог интегрируется с SourceCraft, обеспечивая продвинутые функции анализа кода, проверки качества и соблюдения стандартов в проекте.
