# 🤖 SourceCraft Agent Skills & Configuration

Центральная конфигурация для ИИ-ассистента проекта. Определяет его навыки, правила, поведение и обучение.
**Статус**: 🟢 Active | **Роль**: Source of Truth для AI-поведения в репозитории.

## 📦 Структура

| Файл / Папка | Назначение |
|--------------|-----------|
| `mcp.json` | Главный конфиг Multi-Client Protocol. Запускает агента. |
| `ai-models.yaml` | Настройки моделей (GigaChat, OpenAI, Llama). |
| `custom_modes.yaml` | Режимы: разработка, аудит, релиз. |
| `context.md` | Глобальный контекст, цели и принципы агента. |
| `rules/` | Стандарты кода, безопасности, workflow реализации. |
| `skills/` | 17 специализированных навыков (архитектура, DevOps, SEO и др.). |
| `teacher/` | Гайды, шпаргалки, скрипты автоматизации, шаблоны. |
| `tools/` | Вспомогательные утилиты (RAG-оптимизация, аудит, мониторинг). |

## 🧩 Ключевые навыки (Skills)

| Навык | Описание | Триггер |
|-------|----------|---------|
| `code-security-auditor` | Поиск уязвимостей, проверка секретов | При коммите / по запросу |
| `devops-ci-cd` | Анализ воркфлоу, оптимизация деплоя | При изменении `.github/` |
| `repo-quality-auditor` | Оценка структуры, дубликатов, долга | Еженедельно / по запросу |
| `it-compass` | Применение методологии системного мышления | При проектировании |
| `career` | Анализ прогресса, рекомендации по развитию | По запросу |

> 💡 Полный список: [`skills/`](./skills/)

## 🎯 Назначение
Этот каталог обеспечивает:
- ✅ Единообразие подходов в команде.
- ✅ Автоматизацию рутинных задач и код-ревью.
- ✅ Поддержку высокого качества кода и архитектуры.
- ✅ Быстрый старт новых разработчиков.

## 🚀 Как использовать
1. **Локально в IDE:** Скопируй `codeassistant/` → `.codeassistant/` в корне проекта. Настрой `mcp.json` под свои ключи.
2. **Как шаблон:** Используй `skills/` и `rules/` как референс для своих проектов. Адаптируй `context.md` под свою архитектуру.
