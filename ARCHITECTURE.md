# Архитектура проекта Career Development System

## Обзор архитектуры

Career Development System - это система управления карьерным развитием на основе методологии "Объективные маркеры компетенций". Архитектура построена на микросервисах и клиент-серверном подходе.

## Основные компоненты

### 1. API
- **src/api/**: FastAPI приложение для предоставления REST API

### 2. Веб-интерфейс
- **src/web/**: React приложение для веб-интерфейса

### 3. Мобильное приложение
- **src/mobile/**: React Native приложение для мобильной платформы

### 4. Ядро системы
- **src/core/**: Основные модели и логика системы

### 5. Утилиты
- **src/utils/**: Вспомогательные функции и утилиты

### 6. Тесты
- **src/tests/**: Unit и интеграционные тесты

## Технологический стек

- Python (FastAPI) для API
- JavaScript (React) для веб-интерфейса
- JavaScript (React Native) для мобильного приложения
- SQLite для локального хранения данных (потенциально)

## Интеграция с другими проектами

### Зависимости
- IT-Compass - для получения данных о компетенциях
- Portfolio Organizer - для получения материалов
- Cloud-Reason - для получения инсайтов

### Точки интеграции
- Получение данных о компетенциях из IT-Compass
- Получение материалов из Portfolio Organizer
- Получение инсайтов из Cloud-Reason
- Экспорт данных для интеграции с внешними HR-системами
