# 🏗️ Архитектура Portfolio System Architect

## Общая структура системы

Portfolio System Architect — это система построения и управления карьерой через измеримые навыки, основанная на методологии "Объективные маркеры компетенций". Система состоит из нескольких компонентов:

1. **Cognitive Agent** — автономный AI-агент для оркестрации всей экосистемы
2. **IT Compass** — методология измерения компетенций через маркеры
3. **Микросервисы** — 21+ независимый сервис для различных задач
4. **RAG-система** — для анализа заметок, диалогов и артефактов
5. **Reasoning-модель** — для автоматического сопоставления действий с маркерами

## 4. RAG-система

### Назначение:
- Анализ большого количества заметок, диалогов и артефактов
- Поиск релевантных фрагментов текста по запросам
- Подготовка контекста для reasoning-модели

### Интеграция:
- Сканирует заметки и диалоги (включая файлы из Obsidian)
- Индексирует содержимое для быстрого поиска
- Возвращает релевантные фрагменты по запросу

## 5. Reasoning-модель

### Назначение:
- Анализ результатов RAG-поиска
- Сопоставление действий из текста с маркерами IT Compass
- Генерация отчетов о соответствии

### Процесс:
1. Получает фрагменты текста от RAG-системы
2. Анализирует их на предмет конкретных действий
3. Сопоставляет с маркерами компетенций
4. Формирует структурированный вывод

## Интеграционный процесс

```
Заметки/диалоги (25,000+) → RAG-поиск → Reasoning-анализ →
→ Сопоставление с маркерами IT Compass → Автоматическое портфолио
```

### Этапы:
1. **Индексация**: Все заметки и диалоги индексируются RAG-системой
2. **Поиск**: При запросе находятся релевантные фрагменты
3. **Анализ**: Reasoning-модель анализирует фрагменты и ищет действия
4. **Сопоставление**: Найденные действия сопоставляются с маркерами
5. **Формирование**: Создается автоматическое портфолио с доказательствами

## Когнитивная архитектура

Система превращает неструктурированные заметки в структурированное портфолио, демонстрирует системное мышление через автоматический анализ и показывает путь развития через объективные доказательства.

## Использование системы

### Для базового функционала:

```bash
# Установка
pip install -r requirements-dev.txt

# Запуск тестов
pytest apps/*/tests/ -v

# Запуск агента
python agents/cognitive_agent/orchestrator_v2.py
```

### Для полной системы (с RAG и Reasoning):
1. Загрузить все заметки в RAG-систему
2. Выполнить поиск по ключевым темам
3. Пропустить результаты через Reasoning-модель
4. Сопоставить с маркерами IT Compass
5. Сгенерировать автоматическое портфолио

## Методология "Объективные маркеры компетенций"

Вместо субъективных оценок («знаю Python на 4/5») — объективные, верифицируемые маркеры:
- ✅ «Создал Dockerfile для Python-приложения и запустил контейнер»
- Маркер - это конкретное, проверяемое действие, подтверждённое артефактом
