# IT-Compass

## Описание проекта

Методология объективных маркеров компетенций. Система отслеживания прогресса с авторскими маркерами и интеграцией с портфолио.

## Роль в экосистеме

IT-Compass является ключевым компонентом экосистемы Cognitive-Architecture, предоставляя методологию для отслеживания и развития профессиональных навыков через объективные метрики.
