# 🚀 GigaCode - Включил и работаешь!

## ✅ Автоматическая настройка

### Шаг 1: Запусти автоматическую инициализацию

```powershell
.\scripts\auto-update-gigacode-token.ps1
```

**Что это сделает:**
- Получит Access Token от GigaChat
- Сохранит его в `.vscode/settings.json`
- Создаст кэш токена на 1 час

### Шаг 2: Настрой автообновление (рекомендуется)

```powershell
.\scripts\setup-gigacode-scheduler.ps1
```

**Что это сделает:**
- Создаст планировщик задач Windows
- Токен будет обновляться **каждые 30 минут автоматически**
- Работает в фоне, не мешает работе

### Шаг 3: Перезагрузи VS Code

```
Ctrl+Shift+P → "Developer: Reload Window"
```

**Или просто закрой и открой VS Code**

## 💡 Использование

### Режим Agent (для сложных задач)
```
@agent проанализируй мой код
@agent исправь баг в функции X
@agent создай тесты для модуля Y
```

### Режим Ask (для вопросов)
```
? Что делает эта функция?
? Как работает AI Config Manager?
? Объясни этот код
```

### Inline чат (быстрые правки)
```
Ctrl+I
Напиши: "добавь обработку ошибок"
```

## 🔧 Управление

### Проверить статус
```powershell
.\scripts\GigaCode.psm1
Get-GigaCodeStatus
```

### Остановить автообновление
```powershell
Stop-GigaCodeAutoUpdate
```

### Запустить автообновление вручную
```powershell
Start-GigaCodeAutoUpdate
```

### Проверить подключение
```powershell
Test-GigaCodeConnection
```

## 🐛 Если что-то не работает

### Проблема: GigaCode не отвечает
**Решение:**
```powershell
# Перезагрузи VS Code
# Или:
Ctrl+Shift+P → "GigaCode: Restart"
```

### Проблема: Ошибки подключения
**Решение:**
```powershell
# Запусти заново
.\scripts\auto-update-gigacode-token.ps1
```

### Проблема: Планировщик не работает
**Решение:**
```powershell
# Пересоздай планировщик
.\scripts\setup-gigacode-scheduler.ps1
```

## 🔐 Безопасность

- **Client ID и Client Secret** не хранятся в репозитории
- **Access Token** кэшируется локально (не коммитится)
- **Планировщик задач** работает только на твоём компьютере
- Все секреты в `.gigacodeignore`
