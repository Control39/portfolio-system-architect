# Тестовая задача для демонстрации MCP сервера

## Цель

Продемонстрировать работу интеграции между GIGA IDE и MCP сервером на примере реальной задачи.

## Задача

1. Создать простое Python приложение `hello_world.py` с функцией приветствия
2. Написать тесты для этого приложения
3. Проверить, как MCP сервер применяет правила к процессу разработки

## Ожидаемые результаты

- Приложение должно соответствовать правилам из `rules.json`
- Код должен следовать конвенциям проекта
- Должны быть написаны unit тесты
- Все операции должны проходить валидацию через MCP сервер

## Процесс

1. Создание приложения
2. Генерация кода с применением правил
3. Написание тестов
4. Проверка соответствия правилам
5. Демонстрация работы системы
