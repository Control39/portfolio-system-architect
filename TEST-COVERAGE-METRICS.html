    🧪 Как запустить тесты

    Локально

    # Все стабильные тесты (без покрытия)
pytest apps/it_compass/tests/test_basic.py -v

# С покрытием (для конкретного модуля)
pytest apps/it_compass/tests/test_basic.py --cov=apps/it_compass --cov-report=term-missing

# Полное покрытие (может занять время)
pytest --cov=apps --cov=src --cov-report=html --cov-report=term-missing

    В CI/CD

    - name: Run tests
  run: |
    pytest apps/ --cov=apps --cov=src \
      --cov-report=xml --cov-report=term-missing \
      --cov-fail-under=60  # Временно снижен порог

    📊 Детальные отчёты

    • HTML-отчёт: coverage_html/index.html (открывается в браузере)
    • XML-отчёт: coverage.xml (для CI/CD интеграции)
    • Терминальный отчёт: Выводится после pytest --cov-report=term-missing
