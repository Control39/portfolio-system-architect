## Назначение
- Хранение вспомогательных инструментов для разработки и анализа
- Обеспечение доступа к специализированным утилитам
- Поддержка процессов аудита, мониторинга и визуализации

## Структура
- `ai-agents/` - инструменты интеграции с ИИ
- `ai_integration/` - интеграция с AI провайдерами
- `automation/` - инструменты автоматизации
- `documentation/` - инструменты документирования
- `security/` - инструменты безопасности (security_scan.py)
- `visualization/` - инструменты визуализации
- `dev_tools/` - инструменты разработки
- `debuggers/` - отладчики
- `utils/` - вспомогательные утилиты
- `orchestration/` - оркестрация
- `knowledge-extractor/` - извлечение знаний
- `repo_audit/` - аудит репозитория

Этот каталог содержит специализированные инструменты, которые поддерживают различные аспекты разработки, от анализа кода до визуализации архитектуры, повышая эффективность работы с проектом.

## 📝 Примечания

1. Инструменты в `tools/` предназначены для **локального использования** разработчиками
2. Некоторые инструменты используются **автоматически** в CI/CD или Cognitive Agent
3. Используйте `tools/` для **специализированных задач**, которые не требуют отдельного сервиса
